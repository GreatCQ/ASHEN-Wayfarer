## Part 1：诸神黄昏之后——世界是什么样的

### 1.1 关键前提：黄昏未竟

**诸神黄昏已经发生——但未完成。**

在北欧原典中，诸神黄昏是一次完整的毁灭与重生：世界烧毁、诸神战死、大地沉入海底，然后新的世界从水中升起，幸存的神与人类（Líf 和 Lífþrasir）重新开始。

**但我们的世界不是这样。**

诸神黄昏确实来了。火焰巨人苏尔特挥出末世之剑，九界燃烧。芬里尔吞噬奥丁，耶梦加得毒杀索尔，众神纷纷陨落。世界树的根系被混沌之水侵蚀，开始崩解。

**但就在世界树即将彻底倒塌的那一刻——守根人抱住了它。**

所有守根人，用他们的生命，连接了世界树的断裂根系。Eylíf走得最深——她直接融入了树干，用肉身堵住了最大的裂缝。

世界树没有死。但也没有愈合。它悬停在崩溃与存续之间——像一颗还在跳动的心脏，但每一跳都在失血。

**这就是"黄昏未竟"：毁灭已经发生，但重生没有到来。世界困在了毁灭的半途。**

### 1.2 世界变成了什么样子

诸神黄昏之后的世界，不是废墟——是**伤口**。

| 维度 | 黄昏前 | 黄昏后 |
|------|--------|--------|
| 九界 | 完整连通，世界树承载一切 | 断裂但仍在，世界树勉强维系 |
| 神的权力 | 绝对统治 | 大幅削弱，只能控制自己的域 |
| 人类 | 神的附庸 | 散落各地，自生自灭 |
| 侵蚀 | 不存在 | 世界树伤口持续渗出"混沌之水" |
| 通行 | 世界树根系自由连通 | 根系断裂，域与域之间被侵蚀阻隔 |
| 生态 | 众神维护秩序 | 异兽横行（建木免疫反应失控） |

**关键特征：**

1. **断裂的九界**：九个域还在，但彼此之间的联系被侵蚀切断。从一个域到另一个域，必须穿过侵蚀带——这对普通人来说等于死刑。

2. **渗血的世界树**：世界树没有死，但每时每刻都在"流血"——从伤口渗出的混沌之水不断侵蚀周围的土地。这就是"侵蚀"的物理来源。

3. **神的领域萎缩**：每个幸存的神退守自己的域，用残余的神性维持一个小范围的秩序圈。圈子在缩小——因为侵蚀在扩大。

4. **异兽生态**：建木的免疫系统在试图修复伤口，但反应过度，产生了异兽。异兽不是敌人，是"伤口的痂"——但痂长错了地方，反而成了障碍。

5. **人类碎片化**：没有神的庇护，人类聚居在各个域的边缘，形成小型聚落。有些人继续信神，有些人已经不信了。

### 1.3 什么秩序支撑着世界

诸神黄昏前的世界由**神权秩序**支撑——奥丁的律法、索尔的守护、弗雷的丰收。但那个秩序已经随诸神黄昏一起崩了。

**现在支撑世界的，是两根脆弱的支柱：**

**支柱一：世界树本身**

世界树还在。它的根还在扎在九界的地基里，它的主干还在撑着天穹。虽然它伤痕累累，但只要它还站着，世界就不会彻底崩解。

- 世界树 = 世界的物理地基。它不提供"秩序"，它提供"存在"。
- 只要世界树还在，九界就不会消散。但侵蚀会让它越来越虚弱。

**支柱二：幸存者的固执**

每个域里，都有一个幸存的神在维持秩序。每个聚落里，都有普通人在继续生活。他们不知道明天会不会更好，但他们还在这里。

- 这种"还在"本身就是秩序。不是因为法律，是因为惯性。
- 就像灾后的城市——政府没了，但人们还在废墟上摆摊、做饭、带孩子。

**关键设计点：这两根支柱正在互相矛盾。**

神在维持秩序的方式，是**从世界树汲取力量**。每个神都这样做——我需要力量守护我的域，所以我从世界树多取一些。但世界树本来就快死了，越被汲取就越虚弱，侵蚀就越严重。

**这就是恶性循环：神越守护，世界树越虚弱，侵蚀越严重，神越需要更多力量守护——直到世界树彻底崩溃。**

### 1.4 诸神在做什么

诸神黄昏后，幸存的神各自退守自己的域。他们不是在"统治"——他们是在**困守**。

| 神的状态 | 具体表现 |
|---------|---------|
| 恐惧 | 目睹了同袍的陨落，对侵蚀有本能的畏惧 |
| 困守 | 用残余神性维持一个"安全圈"，圈子在缩小 |
| 排他 | 不信任其他神——诸神黄昏的背叛记忆犹新 |
| 自欺 | 告诉自己"只要守住这里就好"，拒绝面对全局 |
| 矛盾 | 明知道汲取世界树的力量在加速它的死亡，但没有别的办法 |

**每个神都是一座围城——外面是侵蚀，里面是他最后的子民。他们不是坏人，他们是绝望的守城者。**

但他们做的事情，客观上在杀死世界树。

## Part 2：神与守根人——为什么冲突，为什么和解

### 2.1 共同目标

**神和守根人想要同一件事：让世界活下去。**

这不是一个好人和坏人的故事。这是两种"活下去"的方式之争：

```
神的逻辑：
━━━━━━━━━━━━━━━━━━━━━
  世界在崩塌
  → 我必须守住我的域
  → 守住域需要力量
  → 力量来自世界树
  → 我从世界树汲取力量
  → 我在守护子民
  → 这是正确的

守根人的逻辑：
━━━━━━━━━━━━━━━━━━━━━
  世界在崩塌
  → 根源是世界树在失血
  → 必须先止血
  → 止血需要把力量还给世界树
  → 而不是继续汲取
  → 但神不理解这一点
  → 我必须让他们看到
```

**冲突的本质不是善恶，是视角：神看到的是"我的域"，守根人看到的是"整棵树"。**

### 2.2 为什么守根人能走进侵蚀

这是守根人最核心的能力，也是神无法理解的地方：

|  | 神 | 守根人 |
|--|-----|--------|
| 对侵蚀的反应 | 神性被侵蚀消解 → 极度恐惧 | 没有神性可以被消解 → 无所畏惧 |
| 在侵蚀区的能力 | 大幅削弱，可能被侵蚀吞噬 | 能感知侵蚀的"脉搏"，理解它的结构 |
| 根本原因 | 神的力量来自"秩序"，侵蚀是"混沌"——两者互斥 | 守根人的力量来自"修补"——修补不分秩序与混沌，只看"能不能缝上" |

**守根人不是更强，是更简单。神用神性对抗侵蚀，所以被侵蚀克制。守根人用手去缝，所以侵蚀找不到目标。**

这就像扫地僧——他不用内力对抗敌人，他只是在扫地。敌人不知道怎么打一个不反抗的人。

### 2.3 交付逻辑：神为什么跟随守根人

击败神之后，神不是"被收服"，而是**交付**——武侠式的"传功"。

```
击败（战斗阶段）
  → 你打败了神，但你的目的不是杀死他
  → 你的目的是：让他看到你在侵蚀中能做什么
  
证明（叙事阶段）
  → 在战斗中，守根人走进了侵蚀区
  → 守根人在侵蚀中修补了一道裂缝——这是神做不到的事
  → 神第一次看到：有人能在侵蚀里工作
  
醒悟（转折阶段）
  → 神意识到：我的力量来自世界树，我一直在从它身上取血
  → 而这个人是在给世界树输血
  → 我一直在让世界树更虚弱，而他在让它更强壮
  
交付（结果阶段）
  → "我的力量本就来自世界树。现在，把它还回去——通过你。"
  → 神不是臣服，是托付
  → 武侠里的"传功"：前辈把毕生修为传给后辈
  → 不是因为后辈更强，是因为后辈的路更对
  
共鸣（游戏机制阶段）
  → 神以"回声"形态留在守根人身边
  → 守根人成为神之力量的"新容器"
  → 神的战斗意志和领域之力通过守根人的修补之力重新凝聚
  → 这就是"收服神"的叙事包装
```

## Part 3：终章——建木之心的真相

### 3.1 最后的域：建木心

当守根人走过所有八个域，收服了所有神的回声，获得了进入世界树内部的力量——他来到了建木心。

**这里是Eylíf等他的地方。**

在建木心，守根人发现三个真相：

**真相一：侵蚀不是敌人**

侵蚀不是外来的混沌之水。它是**世界树自己的免疫系统在失控**。

就像人体的自身免疫疾病——免疫系统本该保护身体，但它开始攻击自己的组织。世界树在诸神黄昏中受伤太重，它的修复反应进入了过载状态，开始无差别地"修复"一切——包括不该被修复的东西。这就是为什么异兽存在（免疫细胞失控增殖），这就是为什么侵蚀在蔓延（炎症反应扩散）。

**真相二：神的汲取是世界树的病根**

神从世界树汲取力量维持自己，本质上是**在从一个自身免疫患者身上持续抽血**。每抽一次，患者更虚弱，免疫系统更失控。这就是恶性循环的根源。

神不是恶意——他们不知道。他们只知道"我需要力量"，不知道"我的需要正在杀死给我力量的人"。

**真相三：Eylíf在做什么**

Eylíf融入世界树后，她一直在做一件事：**压制世界树的免疫过载**。

她就像一个一直在按住伤口的人——如果她松手，免疫风暴会彻底失控，世界树会在几个小时内死掉。她已经在那里按了很久很久，久到她自己已经快要变成世界树的一部分了。

她听见守根人来了，她说：

> "你终于来了。我快撑不住了。但你来了就好——因为你知道怎么缝。我只是按住，你会缝。放手让我来按，你去缝。"

**Eylíf不是等待救援的公主，她是一个一直在替你扛着门的战友。**

### 3.2 三条路

建木心给出三个选择：

**补天（Mend the Sky）**
- 把所有神的回声力量全部注入世界树
- 世界树的免疫系统恢复正常
- 侵蚀停止蔓延
- 代价：所有神失去神性，成为凡人。他们守护了这么久的"神"的身份，没了。
- 结局画面：一个失去了神力的旧神，和一个守根人，在修复好的世界树下喝茶。很安静。
- 情感：**苦涩的温柔**——你救了世界，但那些把自己活成围城的人，现在什么都没有了。

**治水（Channel the Waters）**
- 不消灭侵蚀，而是把它导流——让世界树的免疫反应走正确的路
- 侵蚀变成"受控的修复力"，异兽变成有序的生态
- 代价：世界永远不会恢复到诸神黄昏前的样子。它带着疤，但疤也是一种形态。
- 结局画面：一个带着伤痕的世界，侵蚀在缝隙中流淌如溪水。异兽不再攻击人，它们变成了九界的生态守护者。
- 情感：**务实的希望**——不追求完美，只要活着就好。

**蜕变（Metamorphosis）**
- 守根人自己融入世界树，和Eylíf一起
- 两个人的修补之力 + 所有神的回声 = 一种全新的力量
- 世界树蜕变成新的形态——不再是旧有的九界结构，而是一种人类和神从未见过的东西
- 代价：守根人和Eylíf不再是"人"。他们变成了世界树的一部分，永远。但世界树变成了一种新的生命——它不再是需要修补的伤者，它自己在生长了。
- 结局画面：世界树发出新的光芒，它的根系向九界延伸出从未有过的纹路。守根人和Eylíf的声音在根鸣中回响——"我们在这里。"
- 情感：**壮烈的升华**——你放弃了自我，但你成为了比自我更大的东西。

- **空洞骑士**的NPC以"怪诞、孤独、环境感"见长，对话极简但氛围极强

- **ASHEN**的NPC应该走"武侠群像"路线——每个NPC都像武侠小说里的人物，出场不多但一出场就有味道，有来路有去处，有执念有软肋

## Part 5：叙事结构与区域规划

### 5.1 结构哲学

用户偏好：**传统线性叙事的骨架 + 区域间的自由探索**

参考原型：
- **艾尔登法环**：开放世界但叙事方向明确，有"推荐路线"但不强制
- **空洞骑士**：银河城式自由探索，但有"主线钥匙"引导
- **奥日2**：强叙事驱动，区域有推荐顺序但允许少量回溯

**ASHEN的叙事结构设计原则：**

1. **大方向清晰**：玩家永远知道"下一步应该去哪"，但可以选择"现在不去"
2. **区域有门槛**：不是所有区域一开始就能进，需要前置条件（能力/钥匙/神的回声数量）
3. **并行区有差异**：并行区域之间难度接近但体验不同（一个偏战斗、一个偏探索、一个偏叙事）
4. **线性段有节奏**：线性段放在情感爆发点，确保叙事冲击力
5. **回溯有价值**：获得新能力/新同伴后回溯旧区域有新内容

### 5.3 每幕叙事内容

---

#### 第一幕·萌芽：中域（线性教学区）

**主题**：根在叫

**叙事目标**：让玩家理解"我是谁""世界怎么了""我要去哪"

**场景**：中域是世界树的根冠——树干与地面交界的地方。诸神黄昏后，这里成了侵蚀最重的地方，也是守根人最后的据点。Eylíf就是从这里走进世界树的。

**流程**：

| 阶段 | 内容 | 叙事功能 |
|------|------|---------|
| 开场 | 守根人在废墟中醒来。周围是侵蚀区。远处有一棵巨树在流血。 | 建立"废墟中的幸存者" |
| 教学 | 基础移动、攻击、修补之力 | 建立"我的能力是修补" |
| 首遇异兽 | 遭遇枯甲兽（教学Boss），感知"异兽不是敌人，是伤口的痂" | 建立"战斗=理解" |
| 听见根鸣 | 世界树发出一声低鸣——Eylíf的声音 | 建立"有人在叫我" |
| 首遇神 | 遇到域内残存的神的使者——被告知"不要靠近世界树" | 建立"神在阻止我" |
| 首域Boss | 蚀根者——一只被侵蚀扭曲的建木守护兽 | 纯战斗教学，不涉及神 |
| 离开中域 | 守根人修补了根冠的裂缝，打开了通往三个域的路径 | "我能修补，我要去更远的地方" |

**Eylíf的线索**：根鸣声中听到——"我在更深处。快来。"

**枢纽**：中域的守根人圣所 → 游戏的大本营（Hub），所有后续域从这里出发

---

#### 第二幕·行远：南域/东域/北域（并行×3）

**总体主题**：行远——走出自己的域，看到世界的不同伤口

**并行设计原则**：
- 三个域难度接近（约Lv.5-15），但体验侧重不同
- 玩家可以任意顺序完成，但每次只能选一个域
- 完成一个域后回枢纽，另外两个域可选
- 必须全部完成才能进入第三幕
- 三个域的完成顺序影响部分对话和细节，但不影响主线

**南域·万灵之巢 — "生之妥协"**

| 维度 | 内容 |
|------|------|
| 主题 | 生命在侵蚀中如何延续 |
| 视觉 | 巨大的根系洞穴，被荧光菌类覆盖，有微弱的生物光 |
| 神 | 繁花母——生命女神的残余。她用自己的身体包裹了整片森林，让万物在她的体内继续生长 |
| 神的逻辑 | "我把一切抱在怀里，它们就安全了"——但她的拥抱太紧了，森林正在被她窒息 |
| 玩家感受 | 美丽但压抑——到处是生机，但生机是畸形的 |
| Boss战 | 繁花母是区域本身——你打的不是一个人，是一个活的森林 |
| 交付 | "我抱得太紧了。你来吧——你知道什么时候该放手。" |
| 回声能力 | 生命之触——恢复型技能，短时间内持续回血 |
| 域内NPC | 一个被繁花母"保护"到变异的村民，请求你让他出去 |
| Eylíf线索 | 在森林深处，发现Eylíf曾来过这里——她留下了一段根须，上面刻着："痛就是生长。别怕。" |

**东域·铁与烬 — "锻与毁"**

| 维度 | 内容 |
|------|------|
| 主题 | 创造与破坏的循环 |
| 视觉 | 巨大的锻炉废墟，岩浆河，金属结构被侵蚀扭曲成怪异的形状 |
| 神 | 炉心兽——锻造之神的残躯。他把自己变成了一个永不熄灭的炉子，用侵蚀本身做燃料来锻造武器 |
| 神的逻辑 | "只要我还在锻造，我就在创造——创造就是抵抗"——但他的锻造加速了侵蚀的扩散 |
| 玩家感受 | 热烈但危险——到处是创造的力量，但创造正在消耗世界 |
| Boss战 | 炉心兽在锻造过程中与你战斗——你必须在铁水飞溅中寻找攻击窗口 |
| 交付 | "我用错燃料了。你来吧——你知道用什么来锻造才不会烧掉整棵树。" |
| 回声能力 | 锻造之击——重击型技能，破防并造成持续灼烧 |
| 域内NPC | 一个被遗弃的锻匠学徒，仍在试图用残火打造一把剑 |
| Eylíf线索 | 在炉心深处，发现Eylíf的一段回忆：她在这里学过锻造——"每一锤都是缝一针。" |

**北域·冰裂之渊 — "寒中裂"**

| 维度 | 内容 |
|------|------|
| 主题 | 秩序与裂变的对峙 |
| 视觉 | 冰封的裂谷，冰面下是深不见底的侵蚀深渊。冰面随时可能碎裂 |
| 神 | 冻息——冬之女神的残余。她把整片领域冻成了冰，用绝对静止来阻止侵蚀扩散 |
| 神的逻辑 | "只要不变化，就不会崩坏"——但她的冻结也杀死了所有活物 |
| 玩家感受 | 壮美但死寂——到处是完美的冰雕，但冰雕都是曾经活着的东西 |
| Boss战 | 冻息制造暴风雪视野受限，你需要修补冰面裂缝来制造攻击平台 |
| 交付 | "我冻结了时间，但时间不该被冻结。你来吧——让这里再动起来。" |
| 回声能力 | 凛冬之壁——防御型技能，创造冰墙格挡 |
| 域内NPC | 一个被封在冰里的人——解冻后发现他还活着，但请求你重新冻住他（"外面太痛苦了"） |
| Eylíf线索 | 冰层最深处，发现Eylíf的脚印——她走过这里时，冰面没有碎。"她太轻了，轻到连冰都感觉不到她在走。" |

---

#### 第三幕·渡劫：海内域（线性解锁）

**主题**：渡

**进入条件**：收集3个神的回声

**叙事功能**：**中点真相揭示**——改变玩家对整个世界的理解

**场景**：海内域是九界的脐带——世界树与深渊之间的过渡地带。诸神黄昏时，这里是最先被淹没的地方，现在是一片沉没的废墟，水下是混沌的侵蚀，水面上是残存的遗迹。

| 阶段 | 内容 | 叙事功能 |
|------|------|---------|
| 入域 | 3个神的回声联手打开了海内域的通道 | "神的力量正在通过你运作" |
| 水下探索 | 在沉没的城市废墟中穿行，修补水下根须 | 世界更深了——问题也更深了 |
| 中点揭示 | 发现建木的"病历"——侵蚀不是外来入侵，是世界树自身的免疫过载 | **玩家认知反转**：我们一直在打的东西不是敌人，是树的求救 |
| Boss战 | 潮回——海之女神的残余。她把自己变成了潮汐，试图把侵蚀"冲走"，但潮汐本身就是侵蚀扩散的载体 | 循环悲剧：越是努力冲走，越是扩散 |
| 交付 | "我以为我在清洗，但我只是把脏水搅到了更多地方。你来吧。" | 神也犯了错——不是恶意，是不理解 |
| 回声能力 | 潮汐之力——机动型技能，短距离瞬移+范围减速 |
| 域内NPC | 一个住在沉船里的老者，声称在诸神黄昏前就住在这里。"我见过这个世界最好的样子。别让它死。" |

**中点之后的叙事变化**：

从海内域开始，玩家知道了一个关键信息：**侵蚀不是敌人。** 这改变了战斗的叙事意义——你不是在"消灭"敌人，你是在"安抚"一个失控的免疫系统。异兽不是敌人，是"生病了的白细胞"。

这个认知会让后续的战斗和交付更有情感重量。

#### 第四幕·入渊：西域/幽域（并行×2）

**进入条件**：收集4个神的回声（海内域的潮回 + 前三域之一）

**总体主题**：入渊——向更深的伤口走去

**西域·旧神废墟 — "旧日不再"**

| 维度 | 内容 |
|------|------|
| 主题 | 旧秩序的崩塌与不可逆 |
| 视觉 | 诸神黄昏的主战场遗迹——巨大的神尸散落在废墟中，被侵蚀覆盖 |
| 神 | 废铸——战争之神的残余。他杀死了入侵者，但杀戮的怒火失控，他把自己也烧了进去 |
| 神的逻辑 | "我必须更强，才能保护所有人"——但力量本身就是腐蚀 |
| 玩家感受 | 悲壮但窒息——到处是英雄的尸体，英雄主义本身是问题的一部分 |
| 交付 | "我以为力量能解决一切。你来吧——你比我强，不是因为你更能打，是因为你更能忍。" |
| 回声能力 | 狂战之怒——爆发型技能，短时间大幅增幅攻击但持续掉血 |
| Eylíf线索 | 在旧神的尸体旁，发现Eylíf在这里修补过——"她不是来战斗的。她是来收尸的。" |

**幽域·暗月之庭 — "直面自己"**

| 维度 | 内容 |
|------|------|
| 主题 | 神的两面——光明与黑暗不可分 |
| 视觉 | 一半是月光花园、一半是侵蚀深渊，中间是一道薄如纸的边界 |
| 神 | 双面——双面神。她的左脸是慈悲，右脸是暴怒。她用分裂自己来同时守护和战斗 |
| 神的逻辑 | "一面守住人，一面打退侵蚀"——但分裂让她越来越不稳定 |
| 玩家感受 | 诡异但迷人——美和恐怖之间只有一步 |
| 交付 | "我不该把自己撕成两半。你来吧——一个人可以同时是慈悲和锋利。" |
| 回声能力 | 双面之镜——切换型技能，防御形态与攻击形态一键切换 |
| 域内NPC | 双面神的侍女，只有一个脸但能模仿任何表情。"你们都想要我笑，但没人问我哭的时候像什么。" |

### 5.4 隐藏域

| 维度 | 内容 |

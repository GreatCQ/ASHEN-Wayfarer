# ASHEN · 远行者 (ASHEN-Wayfarer)

横版共斗 ACT RPG ｜ 北欧神话为骨 · 山海经为皮 ｜ 买断制全平台

> 一句话：你是一个战死的勇士，本该进英灵殿，但诸神黄昏来了，殿塌了。你的灵魂卡在生死之间，不找神解约就会被侵蚀抹除——于是你踏上诸神之路，战胜每一个神，夺取圣物，让它们幻化成"神影"为你而战。

---

## 📁 仓库结构

| 目录 | 内容 |
|------|------|
| **`design-docs/`** | 全部游戏设计文档（GDD、世界观、Boss体系等）。详见 [design-docs/README.md](design-docs/README.md) |
| **`tools/`** | 辅助脚本（Git 一键同步 / 回档） |
| *（根目录）* | 预留给后续项目工程（UE5 工程、源码、资产等） |

> 本仓库当前以**设计文档**为主。后续将兼作**项目工程仓库**，工程代码进入时会建立独立的工程目录（如 `Project/`、`Source/` 等），与 `design-docs/` 并列、互不干扰。

---

## 🚀 快速开始

- **看设计：** 直接进 [`design-docs/`](design-docs/)，从主基线 `ASHEN远行者_统一GDD_v4.1` 看起。
- **同步改动到 GitHub：** 双击运行 `tools/git-sync.bat`（Windows）或 `tools/git-sync.sh`。
- **版本回档：** 运行 `tools/git-rollback.bat`。

## 二、ASHEN远行者怪物体系分层

我们的游戏是**银河城探索 × MH复刷**的杂交品种，怪物需要同时服务两个系统：

```
┌─────────────────────────────────────────────┐
│  大Boss（章节Boss / 域主）                    │
│  = 银河城的门槛Boss + MH的复刷大怪            │
│  首杀推进叙事，复杀获取记忆碎片 + 侵蚀度变化     │
├─────────────────────────────────────────────┤
│  精英异兽（亚种 / 变异体）                     │
│  = MH的亚种生态 + 侵蚀度驱动的形态变化          │
│  不推进叙事，但提供核心复刷材料和装备进阶         │
├─────────────────────────────────────────────┤
│  异兽（区域常规怪）                            │
│  = 银河城的区域敌人 + 建木免疫反应的视觉叙事      │
│  提供探索节奏、资源积累、世界氛围                │
├─────────────────────────────────────────────┤
│  侵蚀体（侵蚀度升高时出现的环境异变）            │
│  = 动态难度调节器 + 世界状态的可视化反馈          │
│  侵蚀度 >60% 才出现，100%时覆盖全区域            │
└─────────────────────────────────────────────┘
```

### 3.1 总量推荐

| 指标 | 基础版 | 含DLC(3个) | 依据 |
|------|--------|-----------|------|
| 大Boss（域主） | 14 | 17 (+3 DLC域主) | MV中位数上沿 × 双主角路径 |

### 3.4 精英异兽配置逻辑

精英异兽不是大Boss的数值翻版，而是**侵蚀度系统驱动的生态变体**：

| 变体类型 | 触发条件 | 设计功能 | 示例 |
|---------|---------|---------|------|
| **侵蚀亚种** | 侵蚀度 >40% | 同一异兽的侵蚀版本，招式变凶猛 | 枯甲兽→蚀甲兽（新增冲撞+毒雾） |
| **区域亚种** | 不同域的同源异兽 | 用同一骨架换皮，降低美术成本 | 南域藤蔓 vs 东域铁蔓（同一动画集） |
| **高阶个体** | 侵蚀度 >80% | 固定位置出现的稀有精英 | 侵蚀体·Prime（掉落高品质碎片） |

18只精英异兽的拆分：
- 侵蚀亚种 × 10（从10只常规异兽衍生）
- 区域亚种 × 5（同源异域换皮）
- 高阶个体 × 3（稀有精英）

## 四、复刷深度设计——从MV到MH的桥梁

### 4.1 银河城游戏的Boss通常不复刷，为什么我们要？

| 游戏的复刷模式 | 玩家动机 | 留存效果 |
|--------------|---------|---------|
| Hollow Knight | 纯挑战（梦魇之王） | 短期，仅限硬核玩家 |
| Dead Cells | Roguelite随机 | 长期，但Boss叙事权重低 |
| MH World | 装备驱动 | 极长期，核心玩法 |

**ASHEN远行者的解法：侵蚀度系统 + 记忆碎片**

复刷不是"再打一次同样的Boss"，而是：
1. **侵蚀度变了** → Boss招式表变了（40%/60%/80%三档变体）
2. **记忆碎片** → 每次击杀获得不同层碎片，拼出Boss的完整故事
3. **装备进阶** → 碎片是锻造材料，驱动MH式的装备追求
4. **叙事层递** → 五层碎片 = 五次对话，Boss从"障碍"变成"角色"

### 4.2 单只Boss的复刷生命周期

```
首杀（叙事体验）
  → 获得残影碎片（灰色）+ Boss基础武器图纸
  → 解锁：此域Boss竞技场直达

复杀1-2次（侵蚀初变）
  → 侵蚀度40%+触发：Boss新增1-2招侵蚀技能
  → 获得骨骼碎片（白色）+ 进阶材料

复杀3-5次（深度侵蚀）
  → 侵蚀度60%+触发：Boss新增侵蚀阶段（P2/P3）
  → 获得心跳碎片（红色）+ 稀有锻造材料

复杀5-8次（记忆回声）
  → 侵蚀度80%+触发：Boss进入"回声"模式（全招式+AI增强）
  → 获得回声碎片（绿色）+ Boss套装核心件

终极挑战（侵蚀巅峰）
  → 侵蚀度95%+触发：Boss"侵蚀体"形态（完全变异）
  → 获得侵蚀碎片（黑色）+ 终极武器/外观
  → 收齐五层碎片 = 解锁Boss完整故事动画
```

**设计目标：每只Boss 6~8次有效复刷，14只大Boss × 6 = 84次Boss战 → 核心战斗内容量对标MH**

### 4.3 与MH对比的复刷效率

| 指标 | MH World | ASHEN远行者 | 说明 |
|------|---------|-----------|------|
| 基础大怪 | 37 | 14大Boss + 18精英 = 32 | 数量接近 |
| 单怪平均猎杀次数 | 15~25次 | 6~8次 | 我们单怪深度浅，但叙事补偿 |
| 装备驱动 | 5件套 × 多武器 | 碎片锻造 + 技能树 | MH的装备深度我们用碎片+侵蚀度替代 |
| 总战斗时长 | ~200h | ~80~120h | 买断制合理区间 |
| 叙事回报 | 几乎无 | 五层碎片 = 完整故事 | **核心差异点** |

## 六、美术资产可行性评估

### 6.1 2D横版的成本优势

| 资产类型 | 3D(MH)成本 | 2D横版成本 | 节省比例 |
|---------|-----------|-----------|---------|
| 大Boss动画 | ~300帧/只 | ~120帧/只 | 60% |
| 精英异兽 | ~200帧/只 | ~60帧/只（换皮+新增） | 70% |
| 常规异兽 | ~150帧/只 | ~40帧/只 | 73% |

14大Boss × 120帧 + 18精英 × 60帧 + 39异兽 × 40帧 = **4,500帧精灵动画**（基础版）

对比Hollow Knight约6,000帧精灵 → **完全在2D银河城项目的合理范围内**

### 6.2 侵蚀度变体的资产策略

侵蚀度变体**不需要重新画整套动画**：
- 40%侵蚀：原精灵 + 色相偏移 + 粒子特效叠加
- 60%侵蚀：原精灵 + 部分帧重绘（新增侵蚀器官/肢体）+ 新增2~3个独立招式帧
- 80%侵蚀：60%版本 + 更多侵蚀特效 + AI增强（不新增帧）

**增量美术成本：每只Boss仅增加约20%的帧数即可支撑三档侵蚀变体**

## Part 0 · 一句话概览 + 设计支柱

### 0.1 一句话概览（Elevator Pitch）

> **《残烬契约 / ASHEN OATH》是一款 2D 横版共斗 ACT RPG。你是一具战死沙场、本该被女武神接进英灵殿的残魂——可就在被接引的途中，诸神黄昏爆发，英灵殿坍塌，你卡在了生与死的夹缝里。契约没有兑现，你既不是人也不是神，是一种不被任何秩序承认的存在，放任下去就会被侵蚀抹除。为了活下去，你踏上诸神之路：找到黄昏中幸存的旧神，夺下他们的圣物之力化作并肩而战的「神影」，一路打回英灵殿的废墟，向当年签下这纸契约的奥丁，讨一个了断。**

**三句话版本**（用于宣发/店面）：
1. **你死了，但天堂塌了，没人来接你。**
2. **不夺神之力，你就会被这个世界删除。**
3. **打穿众神，回到那座废墟，问签约人：为什么。**

### 0.3 设计支柱（Design Pillars）

设计支柱是这款游戏**不可妥协的四条核心体验**。它们是标尺——后续任何一个系统、数值、关卡、剧情决策，都要拿这四条来量：这个设计是在兑现某条支柱，还是在稀释它？**凡是不服务任一支柱、又不可替代的复杂度，一律砍掉。**

每条支柱包含三部分：**玩家必须感受到什么** / **靠什么机制兑现** / **红线（什么情况算这条支柱被破坏了）**。红线是给未来做 Playtest 时的失败判据——先定义"坏"，才认得出"坏"。

---

#### 🗡️ P1 · 以战证道的武侠感

- **玩家必须感受到**："我一个卡在生死间的残魂，没有天赋、没有血统，是凭本事一刀一刀把神打服的。"
- **靠什么机制兑现**：神的力量必须**打赢才拿得到**（圣物→神影）。以战为主，剧情赠予极少（≤2 次，且集中在教学首神与中点传功）。神影是"我打服的对手来给我并肩作战"，不是天降神通。
- **🚩 红线**：
  - 任何一个神影，玩家是靠"刷等级/氪时长自动获得"而不是靠一场有分量的战斗获得 → 破坏 P1。
  - 战斗胜利来自数值碾压而非操作与理解（读招、找窗口、切神影） → 破坏 P1。

#### 🔥 P2 · 逆天改命的悲怆

- **玩家必须感受到**："天堂塌了、契约废了、连'我该去哪'都没了——但我还在往前走。"
- **靠什么机制兑现**：三层契约的**生存倒计时**（侵蚀度）是一直悬在头上的钟；主角始终被称作"掠神者/不被承认者"，直到终章才被重新命名。推进世界的每一步都伴随代价与失去。
- **🚩 红线**：
  - 侵蚀/倒计时被做成纯粹的数值惩罚，玩家只觉得"烦"而非"命悬一线的悲壮" → 破坏 P2。
  - 主角变成"天选之子/血脉觉醒"式的爽文强者，失去了"残魂逆命"的底色 → 破坏 P2。

#### 💎 P3 · 收集即成长的可视化

- **玩家必须感受到**："我想要那个神影，所以我愿意再打一次。"——成长目标看得见、摸得着、可以在角色身上展示。
- **靠什么机制兑现**：圣物（物理神器）→ 神影（可视化战斗形态）→ 战斗风格切换，三段式全部可视。进度条明确：**神器收集 9/9**。复刷有叙事温度（Boss 记忆碎片），不是干刷数值。
- **🚩 红线**：
  - 击败神后拿到的是一串看不见的被动数值，而不是一个看得见、能召唤、能展示的神影 → 破坏 P3。
  - 复刷只为堆数字、没有任何叙事/形态增量 → 破坏 P3。

#### ⚔️ P4 · 两条成长线各司其职的清晰

- **玩家必须感受到**："要新技能，我去打神；要更强的武器装备，我去狩猎异兽。"——两个目标永远不混淆。
- **靠什么机制兑现**：**神 Boss = 技能线**（击败获圣物→神影→技能，紧贴主线叙事）；**异兽 Boss = 装备线**（狩猎获素材→合成武器装备，怪物猎人式共斗底座）。两线物理分离，产出物不交叉。
- **🚩 红线**：
  - 玩家搞不清"我现在该去打神还是该去刷异兽"，两条线的产出物开始互相替代 → 破坏 P4。
  - 装备线沦为可有可无的附庸，或技能线被素材合成绑架 → 破坏 P4。

### 0.4 支柱如何指导决策（使用说明）

| 场景 | 用支柱怎么判 |
|------|------------|
| 有人提议"加一个抽卡获得神影" | 违反 **P1**（不是打赢拿的）+ **P3**（削弱可视收集的分量）→ 否 |
| 有人提议"侵蚀度满了直接删档" | 检查 **P2** 红线：这是悲怆还是纯惩罚？若只让人烦躁 → 改为可逆的高压而非删档 |
| 纠结"神影要不要做成纯被动 buff" | 违反 **P3**（要可视化形态）→ 做成形态切换而非隐藏数值 |
| 纠结"异兽要不要也掉技能" | 违反 **P4**（两线混淆）→ 异兽只掉素材，技能只来自神 |

> **一句话**：P1/P2 是**情绪内核**（为什么这游戏动人），P3/P4 是**系统骨架**（为什么这游戏耐玩）。四条同时成立，这款游戏才成立。

## Part 0.5 · 命名与称呼体系

> 本节确立游戏名方向，以及 NPC 对主角的称呼体系。称呼不是装饰——在碎片化叙事的游戏里，**称呼本身就是世界观的压缩包**（参考黑魂"不死人"、艾尔登"褪色者"）。

### 0.5.1 游戏命名方向

四个核心意象，对应四种命名气质：

| 方向 | 意象 | 扣住的设定层 | 气质 | 候选名 |
|------|------|------------|------|--------|
| ① 灰烬 | 世界烧尽后的残温 | 黄昏未竟 / 英灵"已死" | 悲怆、残温 | 余烬之誓 / 灰烬远行 / **ASHEN OATH** |
| ② 契约 | 卡在生死之间、悬而未决 | 第一层·契约（最独特卖点） | 宿命、未决 | 未竟之誓 / 残契 / Unkept Oath |
| ③ 根系 | 宇宙树 / 生命 | 第二层·Hoddmímir | 宏大、诗意 | 最深的根 / 根之黄昏 / Hoddmímir |
| ④ 神影 | 斩神 / 收集神力 | 第三层·圣物神影 | 动作、对抗 | 神影 / 斩诸神 / Godshade |

**推荐主名（待最终确认）：**

> **《残烬契约》 / ASHEN · OATH**

- 保留 **ASHEN** 已积累的项目辨识度。
- **契约 / OATH** 直指第一层最独特的核心钩子（签约却未被接引、不解约就消失，见 Part 1）。
- "残烬" = 灰烬意象 + "残"字暗示契约未竟、世界未完成，一词三关。

### 0.5.2 主角 4 方称呼体系（确定）

主角没有固定名字（玩家自定义），但**不同立场的人用不同称呼来叫他**。全游戏只保留 **4 方**称呼，避免混乱。这四个称呼像四面镜子，玩家从中拼出"我到底是谁"。

| # | 谁在叫 | 称呼 | 态度 | 何时启用 | 玩家感受 |
|---|--------|------|------|---------|---------|
| 1 | **旁白 / 系统 / 通用**；Hub 锻匠 / 记录者 | **远行者** | 中性·官方 | 叙事开局即用，全程基准称呼 | 这是我的官方身份，有人当我是人 |
| 2 | **失去神力的旧神**；侵蚀体 / 敌对势力 | **掠神者** | 敌意·指控 | 你开始夺神之力后 | 他们恨我、误解我，凭什么 |
| 3 | **同命英灵（先行者）** | **同行者** | 共情·惺惺相惜 | 遇到同样卡在契约里的人 | 我不是一个人 |
| 4 | **Eylíf（爱人）** | **本名** | 私密·唯一 | 全程，唯一不变 | 只有她记得"我"是谁 |

**设计意图（四面镜子）：**

```
中立者叫你「远行者」  → 你是个赶路的人
敌人叫你「掠神者」    → 你是个抢夺者
同类叫你「同行者」    → 你是我们中的一个
爱人叫你「本名」      → 你还是你
```

**关键张力 ——「掠神者」：** 这是贬义的指控，但玩家心里清楚"我夺神之力是为了活、为了救世界树"。这种**被误解**的张力，正好喂养 **P2「逆天改命的悲怆」**——你做的是对的，但世界给你扣了顶恶名，你还是要走下去。

**成长弧线的可视化：** 称呼可随进度演变，作为情感回报——

| 阶段 | 通用称呼演变 | 含义 |
|------|------------|------|
| 开局 | 远行者 | 一个赶路的死人 |
| 中期 | 远行者（旧神改口"掠神者"） | 你开始撼动旧秩序 |
| 终章 | 守护者 / 接根者 `[PLACEHOLDER]` | 你成为世界树新的守护者（终章命名随结局分支，见 Part 11） |

> **唯一不变的是 Eylíf 始终叫你本名**——在你被世界反复重新定义的过程中，她是那个"还记得你本来是谁"的锚点。这条线在终章"你守够了。换我。"达到情感顶点（见 Part 10 · Eylíf / Part 11 · 真结局）。

### 0.5.3 称呼体系服务哪条支柱（对齐 Part 0）

| 称呼机制 | 服务的支柱 | 如何兑现 |
|---------|-----------|---------|
| 「掠神者」的敌意指控 | **P2 逆天改命的悲怆** | 被误解仍前行，强化"残魂逆命"底色 |
| 「本名」这一唯一锚点 | **P2 情绪内核** | Eylíf 是主角人性未被侵蚀抹除的证明 |
| 称呼随进度演变（远行者→守护者） | **P3 收集即成长的可视化** | 成长弧线不只体现在数值，也体现在"世界怎么称呼你" |

> **一句话**：称呼系统是低成本、高回报的叙事杠杆——不占玩法资源，却让玩家在每一句对话里都能读到"我此刻在世界中的位置"。

## Part 1 · 主角身份与三层契约

### 1.1 身份一句话

> **「我战死沙场，本该去英灵殿——但英灵殿塌了。我卡在生死之间，必须找神解约，否则连灵魂都会被侵蚀掉。」**

- **身份**：英灵（Einherjar）· 契约未竟者
- **自定义**：性别、外貌、生前职业（农民 / 铁匠 / 商人 / 流浪者…）由玩家选择
- **零解释成本**：玩家听到"战死的战士"立刻有画面，不需要先理解世界树是什么

> **设计取舍：** 这个身份是从多轮迭代里筛出来的——曾试过"守根人""侵蚀遗存"等更概念化的方向，都因为"玩家第一时间听不懂"被否决。「战死的英灵」胜在**一句话即入戏**：北欧文化里"战死进英灵殿"是常识级认知，反转"殿塌了没接到你"立刻制造出反差与代入。

### 1.2 三层契约叙事（本作核心叙事引擎）

主角的动机分三层，由浅入深，每一层都同时是**叙事钩子**和**玩法压力源**。这是整款游戏"为什么要打神"的完整回答。

#### 第一层：契约 —— 灵魂的生存压力（驱动整个玩法循环）

```
英灵契约的规则：
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
战死 → 女武神选中 → 签订英灵契约 → 进入英灵殿为神效力
                         │
                         ▼
              签约时是「灵魂状态」
              既不是人，也不是神
              是一种"不被认可的中间态"
                         │
              放任不管 → 被侵蚀消解 → 彻底不复存在
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**关键设定：**
- 主角签了英灵契约，但**诸神黄昏来了，英灵殿在他被接引前就塌了**。
- 契约卡在中间态：**签约方（神）无力履约**——英灵殿没了，"接你进殿"这一条永远无法兑现。
- 灵魂状态不被任何秩序认可：英灵殿不收（塌了），Hel 不收（你是战死的，不归她管），人间回不去（你已经死了）。
- **不被认可 = 被侵蚀的合法目标。** 侵蚀会一点点抹掉这个"系统错误"般的灵魂。
- **解药**：只有找到神、与神战斗、夺取神的力量，才能维持自身存在不被侵蚀；只有诸神（或世界树本身）才能终极解除这个契约。

> **这一层把"找神"从情绪驱动升级成生存驱动：不是"我想找神算账"，是"不找神我就消失"。** 玩法上，它就是**侵蚀度倒计时**的叙事根（见 Part 2.3）。

#### 第二层：Hoddmímir —— 幸存与觉醒（驱动主角从"求生"到"斩神"）

```
世界树 = 宇宙       每个世界（域）= 星球
                    │
诸神黄昏时，主角的灵魂坠入世界树根系
                    │
        ┌───────────┴───────────┐
        │                       │
   躲入 Hoddmímir          没躲进去
   守住了灵魂              连灵魂也消散
        │
        ▼
在 Hoddmímir 中，主角了解到一个真相：
「神的力量来源于世界树。
  是神之间的争斗，才导致了世界的崩塌。」
        │
        ▼
主角的目标升级：
不只是续命——而是斩断神与世界树的联系，恢复世界树
```

**关键设定：**
- **Hoddmímir's Holt（赫特密米尔森林）** 是北欧原典中 Ragnarök 幸存者的官方避难所——原典里 Líf 和 Lífþrasir 就藏在这里靠晨露幸存，成为新世界始祖。**神话根基扎实，且与项目旧名 Lífþrasir 形成呼应。**
- 主角不是天选，是**恰好坠入了根系、恰好躲进了 Hoddmímir** 的幸存者。物理原因，不是意志原因（这一条守住 P2 红线：主角不是"天选之子"）。
- 引主角进 Hoddmímir 的，是先他一步坠入、先他一步觉醒的英灵 **Eylíf**——也是他生前的爱人（详见 Part 10）。
- 在 Hoddmímir 里，主角接触到世界树与神之间的能量真相，认知从"我要活"升级为"我要救这棵树"。

> **这一层给主角"斩神"提供了道义理由：不只是为自己续命，是替整个世界止血。** 它同时是 Hub（大本营）的所在地（见 Part 8）。

#### 第三层：圣物 / 神影 —— 力量的获取（驱动战斗与复刷）

```
每个神的能力，都源于一件「圣物」
（如雷神之锤、奥丁之矛——能力的实体载体）
        │
        ▼
战胜神 → 夺取圣物
        │
        ▼
圣物幻化成具体的人物形象 ——「神影」
        │
        ▼
神影成为主角战斗能力的一部分
（不是被动加成，是看得见的战斗形态）
        │
        ▼
升级神影 / 解锁新能力 → 需要不停地与神战斗
```

> **这一层把"神变成技能"彻底实体化：圣物是物理对象（能看、能收集、能展示），神影是可视化形象（能战斗、能切换）。** 它是 P1（以战证道）+ P3（可视化收集）两条支柱的机制载体，完整设计见 Part 6。

### 1.3 三层契约如何咬合玩法循环

这是三层契约最关键的价值——**每一层叙事都对应一个玩法机制，叙事动机与玩法压力焊死在一起，玩家推进故事就是在运转核心循环。**

| 层 | 叙事内容 | 对应玩法机制 | 解决的旧设计问题 |
|---|---------|------------|----------------|
| 第一层·契约 | 不解约就被侵蚀消失 | **侵蚀度 = 生存倒计时**，越打神越续命 | 旧版主角动机弱（找人算账），现在是性命攸关 |
| 第二层·Hoddmímir | 神争斗导致崩塌，要恢复世界树 | **Hub 大本营设在 Hoddmímir**，从这里出击各域 | 旧版"守根人"太抽象，现在有具体避难所和清晰目标 |
| 第三层·圣物神影 | 圣物幻化神影为你而战 | **击败 → 圣物 → 神影 → 形态切换 → 复刷升级** | 旧版"神变技能"不具体，现在每一步都有实体 |

### 1.4 动机的一句话骨架

主角"为什么打神"的答案会随剧情放大——**从"为自己续命"→"为世界救树"→"为一个选择"**。这条完整的动机曲线，以及 Eylíf 等每个角色各自的动机弧线，统一在**叙事部分深挖**（见 Part 5 · 章节叙事结构 / Part 10 · NPC 体系），此处不展开。

|------|-----------|---------|
| 契约侵蚀倒计时（第一层） | **P2 逆天改命的悲怆** | 生存压力是"命悬一线仍前行"的机制来源 |
| 打神夺圣物才能续命（第三层） | **P1 以战证道** | 力量必须打赢才拿得到，不是天赋 |

<a name="part-2"></a>
## Part 2 · 世界观：黄昏未竟

> 本节确立游戏所处的世界状态。**与三层契约完全兼容**——它回答的是"主角醒来时，世界是什么样子"。

### 2.1 关键前提：诸神黄昏已经发生，但未完成

火焰巨人苏尔特挥出末世之剑，九界燃烧；芬里尔吞噬奥丁，耶梦加得毒杀索尔，众神纷纷陨落。世界树根系被混沌侵蚀，开始崩解。

**但世界树没有彻底倒下。** 它悬停在崩溃与存续之间——像一颗还在跳动、但每跳一次都在失血的心脏。

> **黄昏未竟：毁灭已经发生，但重生没有到来。世界困在了毁灭的半途。**

这个前提解决了一个关键矛盾——**神既"死"了（原典战死），又"活"着（还能当 Boss 打）**。答案是：黄昏没走完，该结束的没结束，该重生的没重生，所有人都卡在半途（神的具体生死形态见 Part 3）。

### 2.2 世界树 = 宇宙，每个域 = 星球

- 世界树是世界的**物理地基**。它不提供"秩序"，它提供"存在"本身。
- 九个域像九颗星球，挂在世界树这棵"宇宙树"的不同枝干上（九域映射见 Part 4）。
- 域与域之间的联系被侵蚀切断。穿越域间必须穿过侵蚀带——对普通人等于死刑，但对"已经死过一次"的主角不然。这解释了**为什么只有主角能跨域旅行**。

### 2.3 侵蚀的双重含义（一个数值，两层意义）

侵蚀对世界、对主角，是**同一种东西的两个视角**：

| 视角 | 侵蚀是什么 | 后果 |
|------|----------|------|
| 对世界 | 世界树**自身免疫过载**（受伤太重，修复反应失控，无差别"修复"一切） | 异兽横生、域间阻隔、世界树持续失血 |
| 对主角 | 死亡在拉这个"不被认可的灵魂"回去 | 侵蚀度越高越接近彻底消亡，但死亡之力也越强 |

> 这让**侵蚀度系统**既是世界状态条，又是主角的生存倒计时——**一个数值，两层意义**。它是 P2（逆天改命的悲怆）的核心机制载体，也是复刷循环的叙事根（系统细节见 Part 7）。

### 2.4 神在做什么：绝望的守城者

幸存的神各自退守自己的域，不是在"统治"，是在**困守**。

每个神都是一座围城——外面是侵蚀，里面是他最后的子民。他们用残余神性维持一个"安全圈"，而维持的方式是**从世界树汲取力量**。

```
恶性循环：
神越想守护 → 越多汲取世界树 → 世界树越虚弱
     ▲                                │
     └────── 侵蚀越严重 ← 神越需要更多力量 ←┘
```

> **神不是反派，是绝望的守城者。** 但他们做的事，客观上在加速杀死世界树。这正是第 5–6 章**中点反转**的种子——玩家前期以为神是元凶，中点发现"神也是溺水的人，只是他们抢的木板是整个世界的命"（反转节点见 Part 5）。

### 2.5 本 Part 服务哪条支柱

| 世界观设定 | 服务的支柱 | 如何兑现 |
|-----------|-----------|---------|
| 侵蚀=世界失血+主角倒计时 | **P2 逆天改命的悲怆** | 世界和主角同时"命悬一线"，悲怆有世界级共鸣 |
| 神是绝望守城者（非反派） | **P2 情绪深度** | 敌人有苦衷，胜利不轻松，为中点反转铺垫 |
| 只有死过的主角能跨域 | **P1 以战证道** | 主角的"资格"来自处境而非血统，仍要靠打 |

## Part 3 · 神的生死与形态规则

> 本节回答一个必须自洽的问题：**诸神黄昏原典里，很多神明明死了，为什么还能被主角当 Boss 打？** 规则是——**神不会"消失"，只会"换一种存在方式"**。每个神是什么形态，都由他在原典里的真实命运决定，不是随意设定。

### 3.1 核心规则：原典三分法

诸神黄昏原典给了一个干净的三分法，正好覆盖"该死的、该活的、该归来的"三种神：

| 命运类别 | 原典依据 | ASHEN 形态设计 | 击败后处理 |
|---------|---------|--------------|----------|
| **A · 战死系** | 奥丁、索尔、提尔、海姆达尔在原典确实战死 | 肉身已灭，留下**圣物 + 残响形态**（怨念 / 意志凝聚的 Boss） | 残响散去，圣物留下；不强行复活 |
| **B · 幸存系** | 维达尔、尼约德、弗蕾雅（华纳系）在原典幸存 | 活着但虚弱，打**困守形态** | **失去神力但存活** → 转枢纽 NPC |
| **C · 复活 / 异态系** | 巴德尔从冥界归来；赫尔本就是冥界主宰（非死亡对象） | 死后**换形态归来**（幼态 / 灵态 / 平行态），或以冥府之主的异质形态现身 | 异态被收束为圣物；本体或留或去 |

> **玩家的核心诉求"神不要死，失去神力已是最大惩罚"——B 类完全满足；A 类尊重原典（已死的不强救，但用残响让你还能打）；C 类用魔幻设定的"换形态"提供叙事惊喜。三类刻意混合分配到各域，让每打一个神都有新鲜的"神的状态"体验。**

### 3.2 形态设计原则

**为什么"失去神力"就是最大的惩罚？**

神的全部尊严建立在"我是神、我能守护我的域"上。夺走圣物 = 夺走神力 = 他再也守不住任何东西。一个失去神力的旧神在枢纽里坐着，看着主角带着他的力量去做他做不到的事——**这比杀了他更残忍，也更有叙事重量**（对标 ENDER MAGNOLIA 净化后同伴存活的温度）。

**圣物 = 神力的实体钥匙：**
- 每个神的能力都锁在一件圣物里（雷锤、号角、火之项链…）。
- 战胜神 → 圣物易主 → 神力随之转移。这解释了"为什么打赢就能拿到力量"——不是抽象吸收，是**钥匙换了主人**（圣物→神影的战斗系统见 Part 6）。

## Part 4 · 九界映射（北欧骨 + 山海经皮）

### 4.1 映射原则：北欧定题材与结构，山海经供设计灵感

游戏世界由两套神话协作搭建，但两者的**职责层级不同**——北欧是根本的题材与框架，山海经是在这个框架内的设计灵感来源：

| 神话 | 承担什么 | 具体落在哪 |
|------|---------|-----------|
| **北欧神话** | **游戏的题材与时间线主干** | 整个背景是"诸神黄昏"、世界框架是九界、Boss 就是北欧诸神——这是故事、时间线、世界结构的根，不可替换 |
| **山海经** | **各域的设计灵感与映射对象** | 在北欧结构确定后，每个域的视觉、生态、异兽、地貌从山海经取材、映射设计——它是灵感库，参与"这个域具体长什么样、住着什么" |
| **两者交汇点** | **世界树 = 建木** | 北欧伊格德拉希尔与山海经建木本就是同源母题（都是世界中心的通天神树），成为两套神话天然的焊接点 |

**为什么这样分工：**
- **北欧负责"玩家为什么想玩"**——诸神黄昏是全球认知度极高的题材，索尔、奥丁、洛基是玩家听过名字就想挑战的神。题材与神的辨识度，是吸引力的来源。
- **山海经负责"玩家为什么记得住"**——同样是砍神的横版动作，用山海经的异兽、地貌、东方奇幻质感做设计映射，能和市面上"纯北欧魂 like"拉开差异，形成独特气质。

> **一句话：北欧是这款游戏的题材与结构本身；山海经是在这套结构里做具体设计时的灵感来源与映射对象。前者定"讲什么故事、什么世界"，后者定"这个世界具体长什么样"。**

- **域间辨识度递增**：越往后的神越知名（索尔 → … → 巴德尔 → 洛基 → 奥丁），玩家每次都觉得"对手更大牌了"——这是 Boss 排布逻辑的一部分（见 Part 5 · 权力金字塔）。

## Part 5 · 章节叙事结构

> 本节是叙事主干——**五幕九章 + 终章 + 二隐藏章**，以及贯穿全程的动机曲线、中点反转、Boss 排布逻辑。**整合自《章节区域与双线Boss体系》第 4 节，并把 Part 1 精简掉的动机三阶段曲线在此深挖落地。**

### 5.1 主角动机曲线：为什么打神的答案会三次质变

这是维持 30–40h 主线情绪不疲劳的关键。主角"为什么打神"不是一个恒定答案，它随剧情**放大三次**——每次放大都由一场关键 Boss 战承载：

| 阶段 | 章节 | 动机（主角心里的答案） | 玩家情绪 | 对应支柱 |
|------|------|---------------------|---------|---------|
| **① 追债**（为自己） | 第 1–4 章 | "英灵殿塌了没人接我，你们欠我一个交代。我夺你的力，先让自己别消失。" | 愤怒、索取 | P2 悲怆的起点 |
| **② 困惑**（重估） | 第 5–6 章 | "等等——神好像也在等死。他们不是元凶，是溺水的人在抢木板。" | 动摇、震动 | P2 从愤怒转向复杂 |
| **③ 抉择**（为一个选择） | 第 7–9 章 + 终章 | "我手里握着九件神力。我到底要用它——毁掉旧世界、守住它、还是重铸它？" | 决断、承担 | P2 悲怆的落点 |

```
动机放大轨迹：
为自己（续命） ──→ 为世界（救树） ──→ 为一个选择（终章三条路）
   第1-4章            第5-6章               第7-9章+终章
   愤怒               动摇                   决断
     └───────── 每次放大都由一场 Boss 战顶上去 ─────────┘
```

> **设计要点**：动机每放大一次，玩家对"神"的看法就重构一次——从"敌人"到"同类受害者"再到"我必须替他们做个了断"。这条曲线是整个游戏情感重量的来源，也是 P2（逆天改命的悲怆）从起点走到落点的完整路径。各角色（尤其 Eylíf）各自的动机弧线见 Part 10。

叙事采用 **五幕九章**（幕 = 情绪段落，章 = 域）。每章一个域、一个神、一条核心主轴。

```
第三幕 · 渡劫（中点 · 打华纳神族 / 真相反转）

第四幕 · 入渊（后期 · 打核心圈 / 奥丁至亲）

隐藏幕（探索触发）
  ├ 天域：选择不参战的复仇神（维达尔）
  └ 根域：最后的根（真结局 · Eylíf）
```

### 5.4 Boss 排布的三重逻辑（为什么是这个顺序）

Boss 出场顺序不是随意排的，遵循**三条同时成立的原则**——这是"前后出现关系"的核心：

| 原则 | 含义 | 玩家体验 |
|------|------|---------|
| **① 辨识度递增** | 越往后的神越知名（索尔 → … → 巴德尔 → 洛基 → 奥丁） | 每次都觉得"对手更大牌了"，期待感持续走高 |
| **② 权力金字塔** | 从战场边缘神 → 华纳神族 → 奥丁至亲 → 奥丁 | 物理上越打越靠近世界权力核心 |
| **③ 原典因果链** | 每个神的位置扣住他在诸神黄昏的真实角色 | 神话粉会心一笑，普通玩家觉得顺理成章 |

**追债链条（前后关系）：** 主角是债主，要找签约人奥丁。但奥丁在最里面，必须一层层打穿他的人——

```
打他的战士（索尔 / 海姆达尔）
  → 打他借力的华纳盟友（赫尔 / 尼约德）
    → 打他的至亲（爱子巴德尔）= 向奥丁宣战
      → 打表面元凶（洛基）= 识破"放火者不是答案"
        → 够得着奥丁本人 = 终极清算
```

### 5.5 为什么终极 Boss 是奥丁

奥丁是唯一可能的终极 Boss，因为他**同时是主角所有动机的源头**：

| 身份 | 为什么指向他 |
|------|------------|
| **契约签订者** | 英灵契约是奥丁定的规矩。主角契约未履行、卡生死间被侵蚀，债主就是他 |
| **英灵殿主人** | 殿塌了没接走主角，是他失职。终章在英灵殿废墟决战，叙事闭环 |
| **不肯放手者** | 中点真相揭示神在汲取世界树续命，而奥丁是最执着"维持旧秩序、不让黄昏完成"的神。他困住了世界的重生 |
| **反高潮陷阱** | 玩家以为放火的洛基是终极 Boss。打完发现：放火者早疯了，真正该负责的是那个明知世界该重生、却死攥不放的老王。比"打败大反派"更有重量 |

> **一句话：你为奥丁的战争而死，奥丁欠你一个交代，奥丁困住了世界——所有线索的终点只能是他。**（终章战与三结局的详细设计见 Part 11。）

## Part 6 · 圣物 / 神影 战斗系统

> 本节是第三层契约的系统化，也是本作**最核心的差异化卖点**。它是 P1（以战证道）+ P3（可视化收集）两条支柱的机制载体。**内容承接 v4.1 Part 4，并补充机制规格与 `[PLACEHOLDER]` 调参杠杆。**

### 6.1 三段式获取链

```
战胜神  →  夺取圣物  →  圣物幻化「神影」  →  神影成为战斗能力
（战斗）   （实体掉落）   （可视化形象）        （形态切换）
```

### 6.2 神影是什么

**神影 ≠ 被动数值加成；神影 = 一个看得见的战斗形态 / 伙伴。**

切换神影会改变主角的：
- **攻击模组**（连段路线、招式风格）
- **特殊能力**（位移、控场、爆发、恢复…）
- **视觉表现**（神影形象环绕 / 附身 / 召唤）

> **红线守护（P3）**：如果神影做成看不见的被动 buff，这条支柱就破了。神影必须**看得见、能召唤、能展示**。

### 6.3 与竞品的定位区分

| 游戏 | 击败获得 | 表现形式 | 操作方式 |
|------|---------|---------|---------|
| Nioh 2 | 魂核 | 装备槽 | 按键召唤妖怪一次性攻击 |
| ENDER MAGNOLIA | 同伴 | 持续跟随 | 主动技能 + 被动加成 |
| 讨鬼传 2 | 御魂 | 3 槽位 | 映射按键独立施放 |
| Persona | 人格面具 | 整套切换 | 切换整组技能 |
| **ASHEN** | **圣物 → 神影** | **战斗形态切换** | **切换神影改变战斗风格** |

**ASHEN 的独特点：神影做成「形态切换」，与武器系统形成乘法关系。**

```
武器（MH 式基础攻击循环）  ×  神影（连段变体 / 特殊能力）  =  战斗风格

例：
大剑 × 索尔神影   = 雷击附魔型重击流
大剑 × 赫尔神影   = 冰霜控场型减速流
```

> **武器数量可以少（控成本），神影数量多（来自每个神），组合产生深度。这是用最低美术成本撬动最大 Build 深度的关键设计。** 双线成长的完整架构见 Part 7。

### 6.4 神影核心机制（概念）

- **切换即变身**：切换神影 = 换一套攻击模组 + 一组特殊能力 + 一个视觉形态。玩家用切换来应对不同战斗局面（切赫尔控场、切索尔爆发）。
- **有取舍地切换**：神影切换与技能消耗资源、有冷却，不能无脑乱切——具体资源/冷却/无敌帧等数值属于**战斗手感调参**，留待后续专门的战斗数值设计，本 GDD 不展开。
- **神影装备槽建议 3 个**（主战 + 副战 + 被动）——"够搭配、不过载"的甜点区，作为概念结论，具体验证放到原型测试。

> **失败信号（先定义坏）**：如果最终玩家全程只用 1 个神影通关、其余槽位闲置，说明神影同质化或切换成本过高——这是本系统的失败态，需触发平衡调整。

### 6.6 复刷循环：为什么玩家会再打一次

升级神影 / 解锁新能力 = 必须不停与神战斗。这给侵蚀度复刷提供了**叙事 + 机制双重理由**：

```
首杀        → 获得圣物 + 神影（基础形态）
复刷（侵蚀40%）→ 神影解锁第 2 连段变体
复刷（侵蚀60%）→ 神影解锁特殊能力
复刷（侵蚀80%）→ 神影"觉醒形态" + 圣物外观进阶
收齐        → 神影完整故事动画 + 终极形态
```

> 对接 Boss 五层记忆碎片（见 Part 9）——每次复刷不只是数值，是神影故事的逐层揭开。这守住 P3 红线：**复刷有叙事温度，不是干刷数字。**

> **复刷强度是否硬性要求？** 建议**部分可替代**（核心变体靠复刷，纯数值部分可用素材替代升级），避免逼氪式肝——作为概念结论，肝度调参留待后续。

| 武器 × 神影乘法关系 | Build 深度 | 低成本撬动高深度，见 Part 7 |
| 复刷有记忆碎片叙事 | **P3 红线守护** | 复刷有温度，不是干刷 |

## Part 7 · 四层战斗架构 + 双线成长

> 本节是战斗系统的骨架。**四层架构**（武器 × 神影 × 连段 × 锻造）承接 v4.1 Part 5；**双线成长**（技能线 / 装备线）整合自《章节区域与双线Boss体系》第 1 节。它是 P4（两条成长线各司其职）的机制载体。**保持概念层，具体战斗数值留待后续专门设计。**

### 7.1 四层战斗架构

战斗系统由四层叠加而成，从下到上：基础循环 → 形态变体 → 协同连段 → 长线养成。

```
┌──────────────────────────────────────────────┐
│  第 1 层 · 武器（基础循环）                     │
│  MH 式：每把武器 = 一套完整攻击模组 / 节奏       │
│  控成本：武器 4-5 把（每把一整套设定，尽量少）    │
├──────────────────────────────────────────────┤
│  第 2 层 · 神影（形态触发器）                    │
│  圣物幻化，切换改变攻击模组 / 特殊能力            │
│  来自每个被击败的神                              │
├──────────────────────────────────────────────┤
│  第 3 层 · 连段（神影协同）                      │
│  节奏窗口触发：武器普攻 → 窗口内接神影技 → 连段   │
│  多神影组合产生不同连段链                         │
├──────────────────────────────────────────────┤
│  第 4 层 · 锻造（装备养成）                      │
│  圣物碎片 / 异兽素材 → 武器面板 / 神影升级        │
└──────────────────────────────────────────────┘
```

| 层 | 来源原型 | 核心机制 | 玩家决策 |
|----|---------|---------|---------|
| 武器 | MH / Nioh 2 | 基础攻击循环 | 选哪把武器定基础节奏 |
| 神影 | ENDER MAGNOLIA / Nioh 2 魂核 | 形态切换 | 带哪几个神影 |
| 连段 | 湮灭之潮骑士系统 | 节奏窗口触发 | 何时切、怎么接 |
| 锻造 | MH / 讨鬼传 2 | 素材养成 | 强化哪套 Build |

> **连段触发方式：概念定为「节奏窗口」**——武器普攻打出后有一个短暂窗口，窗口内接神影技能可衔接连段。这最适合横版动作的手感表达。具体窗口时长等数值留待战斗数值设计。

> **武器数量：定为 4-5 把（尽量少）。** 每把武器 = 一整套攻击模组、动画、节奏、手感设定，制作成本极高，宁精勿滥。深度不靠"武器多"，靠**武器 × 神影的乘法**撑起来——4-5 把武器 × 9 个神影，理论组合已达 36-45 种战斗风格。这是"少武器、多神影"的核心取舍：把有限的美术产能压在神影的差异化上，而不是摊薄到十几把武器。

### 7.2 双线成长：技能线 × 装备线（P4 的核心）

**两条成长线物理分离，各管一摊，玩家永远清楚"要什么就去打什么"。**

```
线 A · 技能线（紧贴叙事 / 神 Boss）
战胜神 Boss（或剧情赠予）→ 获得神器（圣物）→ 幻化神影 → 复刷升级神影技能
                                        神器 ↔ 神影 ↔ 技能 一一对应

线 B · 装备线（共斗循环 / 异兽 Boss）
战胜异兽（小怪 / 精英 / 兽 Boss）→ 获得素材 → 合成武器 + 装备 → 提升基础面板
                                    独立武器系统（MH 式）

两线在战斗里相乘 = 完整 Build：
武器（线 B 造）× 神影（线 A 得）= 战斗风格
```

### 7.3 两线的职责切分

| 维度 | 线 A · 技能线 | 线 B · 装备线 |
|------|------------|------------|
| 来源 | 神 Boss（章节核心） | 异兽（小怪 / 精英 / 兽 Boss） |
| 产出 | 神器 → 神影 → 主动技能 | 素材 → 武器 + 防具 |
| 决定 | "你会什么神技" | "你的武器面板多硬、节奏如何" |
| 叙事权重 | ★★★★★（贯穿主线） | ★★（玩法循环，弱叙事） |
| 获取方式 | 以战为主，剧情少量赠予 | 纯战斗掉落 |
| 对标 | 洛克人 Boss 武器 / ENDER MAGNOLIA 同伴 | 怪物猎人素材合成 |
| 数量级 | 少而精（9 个神器 / 神影） | 多而广（多套武器 + 防具） |

### 7.4 为什么这个切分好

- **认知零负担**：玩家一眼就懂——想解锁新招式去挑战神，想强化武器去刷异兽。两个目标不会互相干扰（这是 P4 红线：一旦两线产出物互相替代，就破了）。
- **叙事与玩法各得其所**：神 Boss 是"叙事高潮 + 能力里程碑"，刻意做少做精；异兽是"日常循环 + 共斗内容"，可以做多做密。
- **天然支撑共斗 DLC**：线 B（异兽素材合成）就是怪物猎人式共斗的底座，后续多人联机狩猎异兽可无缝接入。
- **神器神影一一对应**：玩家收集 9 个神器 = 集齐 9 个神影 = 9 套神技，收集目标清晰可数，符合心理学的 endowment effect（投入感越强越舍不得放下）。

### 7.5 "剧情赠予"的设计纪律

神力以战为主，但允许剧情少量赠予——**纪律是：赠予只用在"叙事必须、战斗不合适"的节点。**

| 允许赠予的场景 | 例子 |
|--------------|------|
| 首神 · 教学 | 索尔之力作为教学奖励直接给（见 Part 9），让玩家立刻体验"神影"是什么 |
| 牺牲式交付 | 某幸存神自愿把神力传给主角（武侠传功），不打而得，反衬"以战证道" |
| 终章关键 | 终章某神力作为剧情转折点赠予 |

> **赠予 ≤ 2 个**（首神 + 1 个叙事节点）。其余 7 个神器必须靠战胜获得，守住 **P1「以战证道」**内核。

### 7.6 本 Part 服务哪条支柱

| 系统设计 | 服务的支柱 | 如何兑现 |
|---------|-----------|---------|
| 双线物理分离 | **P4 两条成长线各司其职的清晰** | 想要技能打神、想要装备刷兽，永不混淆 |
| 神力赠予 ≤ 2 | **P1 以战证道** | 绝大多数力量靠打赢获得 |
| 武器 × 神影乘法 | Build 深度 | 4-5 武器 × 9 神影 = 低成本撬动 36-45 种战斗风格 |
| 神器 9/9 可数收集 | **P3 收集即成长的可视化** | 目标清晰可数，投入感强 |

## Part 8 · 区域 · 时长 · 关卡配置

> 本节确立游戏体量的宏观框架——时长目标、三层地图结构、各域小区域配置。**整合自《章节区域与双线Boss体系》第 2、5 节。所有时长为设计目标 `[PLACEHOLDER]`，待纸面流程模拟与首测验证。**

### 8.1 时长目标（对标竞品）

| 目标 | 时长 | 对标依据 |
|------|------|---------|
| **主线（打完奥丁）** | **30–40h** | 介于渎神 2（20–30h）与空洞骑士主线（25–35h）之间 |
| **完整体验（含隐藏域 + 真结局）** | **60–80h** | 对标空洞骑士 100% |
| **核心复刷（含神影满级 + 装备毕业）** | **80–120h** | 怪猎式复刷补足长尾 |
| **平均单域主线时长** | **3–4.5h** | 9 主域 × 3.5h ≈ 31.5h 主线核心 |
| **单域完整探索（含支线 / 收集）** | **5–7h** | 含异兽刷取、隐藏路径回溯 |

> **会话假设**：平均玩家单次会话 1–1.5h。每个域设计为"2–3 次会话通主线，再花 1–2 次会话探索 / 刷取"——符合横版银河城的节奏感。`[PLACEHOLDER]` 待首测验证。

### 8.2 三层地图结构

```
世界地图（世界树 · 九域挂在不同枝干）
   └ 域地图（单个域 = 银河城式互联大区）
        └ 小区域（域内功能分区：3-5 个 / 域）
             └ Boss 竞技场（神战场 / 异兽巢穴，MH 式独立场景）
```

- **世界地图**：世界树全景，玩家从 Hub（Hoddmímir 树心）选择前往哪个域。
- **域地图**：单个域是银河城式互联结构，有主路、支路、需要能力解锁的回溯点。
- **小区域**：域内的功能分区（见 8.3）。
- **Boss 竞技场**：神战场 / 异兽巢穴，MH 式独立场景，可复访复刷。

### 8.3 单域内的"小区域"构成

每个域是一个银河城式互联地图，内部切成 **3–5 个小区域**，承担不同功能：

| 小区域类型 | 功能 | 每域数量 | 时长占比 |
|-----------|------|---------|---------|
| **入口区** | 进入域、低难度异兽、建立氛围 | 1 | ~15% |
| **探索区** | 主体探索、平台跳跃、精英异兽、收集 | 1–2 | ~40% |
| **支线区** | 可选路径、隐藏宝箱、异兽巢穴（刷素材） | 0–1 | ~20% |
| **神域核心区** | 通往神 Boss 的路、解谜门槛 | 1 | ~15% |
| **神战场** | 神 Boss 竞技场 | 1 | ~10% |

> **功能分区的意义**：每类小区域对应一种玩家活动（探索 / 战斗 / 解谜 / 刷素材），保证单域内的节奏有起伏，不是一条直路打到底。支线区是装备线（线 B）的刷取场所，与主线神战场（线 A）物理分开，呼应 P4 双线清晰。

### 8.4 各域时长与小区域数（配置总表）

> **均匀原则**：九个主域体量刻意做均匀，主线时长收在 **3–4h** 的窄带内，小区域数在 **4–5 间小幅波动**（而非越到后期越大）。差异体现在**节奏与主题**上，不体现在体量堆叠上——避免后期域过大造成疲劳。教学域（中域）偏短，终章 / 隐域按叙事需要另计。

| 章 / 域 | 主线时长 | 完整探索 | 小区域数 | 节奏定位（差异在节奏，不在体量） |
|-------|---------|---------|---------|---------|
| 1 中域 | 1.5–2h | 2.5h | 3 | 教学，线性偏多 |
| 2 雷域 | 3–3.5h | 5h | 4 | 第一个完整域，建立循环 |
| 3 花域 | 3.5–4h | 5.5h | 5 | 探索密度高（迷宫感） |
| 4 铸域 | 3–3.5h | 5h | 4 | 锻造系统深化 |
| 5 寒域 | 3–3.5h | 5h | 4 | 极简 · 生存压测 |
| 6 海域 | 3.5–4h | 6h | 5 | 中点 · 叙事重 |
| 7 巨域 | 3.5–4h | 5.5h | 5 | 垂直场景（巨人身躯攀爬） |
| 8 幽域 | 3–3.5h | 5h | 4 | 自省 · 解谜偏多 |
| 9 劫域 | 3–3.5h | 5h | 4 | 洛基 · 反高潮 |
| ★ 英灵殿废墟 | 2–3h | 3.5h | 2 | 奥丁 · 终极决战 |
| 隐1 天域 | 1.5h | 3h | 2 | 隐藏 · 叙事秘境 |
| 隐2 根域 | 1h | 2h | 2 | 隐藏 · 真结局 |
| **合计** | **~34h 主线** | **~53h 完整** | **44 小区域** | — |

> 主线 ~34h 落在 30–40h 目标内；完整 ~53h（再加复刷 / 神影满级补足到 60–80h 区间）。九主域主线时长全部收在 3–4h，体量均匀；节奏差异靠**主题与关卡类型**制造（迷宫 / 垂直攀爬 / 极简压测 / 解谜…），而非拉长某个域。`[PLACEHOLDER]` 所有时长待首测校准。

### 8.5 Hub：Hoddmímir 树心避难所

- **位置**：世界树根系深处的幸存者避难所（原典 Ragnarök 幸存地），不计入九域。
- **功能**：出击各域的枢纽、锻造 / 升级、NPC 聚集地、剧情推进点（对标哈迪斯的冥界之家）。
- **演化**：随剧情推进，Hub 会因主角带回的神影 / NPC 而逐渐"活"起来——从荒芜到有生机，本身就是进度的可视化（见 Part 10 NPC）。

### 8.6 本 Part 服务哪条支柱

| 配置设计 | 服务的支柱 | 如何兑现 |
|---------|-----------|---------|
| 单域节奏有功能分区起伏 | 探索体验质量 | 不是一条直路，探索 / 战斗 / 解谜交替 |
| 支线区（刷素材）与神战场分离 | **P4 双线清晰** | 装备线场所与技能线场所物理分开 |
| Hub 随进度"活"起来 | **P3 收集即成长的可视化** | 大本营的变化本身就是成长反馈 |

### 9.6 首神：索尔 · 新手引导落地

> **新手引导后就给玩家第一个神力，这个神 = 索尔（雷神）。**

**为什么是索尔：**
- **全球辨识度第一**：雷神之锤是流行文化符号，第一个神就是索尔，冲击力拉满。
- **雷击爽快感最直接**：雷电的视听反馈强烈，新手第一次用神影就"哇"。
- **原典战死系**：索尔与耶梦加得同归于尽——打的是"雷霆残响"，力量仍在锤中，符合"以战为主 + 教学赠予"的混合。

**新手引导流程（Onboarding）：**

```
中域 · 教学（第1章，1.5-2h）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[0-30s]  核心动作：移动 + 普攻 + 闪避，30 秒内全部 introduce
[1-3min] 第一次战斗：教学异兽（必胜，无失败可能）
[5min]   首次"侵蚀"演出：让玩家看到契约侵蚀在啃噬自己 → 建立紧迫感
[10min]  解谜 / 平台小段：教学探索与回溯
[15min]  遇到索尔的"雷霆残响"——第一场神战（教学 Boss，难度可控）
[20min]  战后拾起 Mjölnir → 神影系统首次解锁 → 立刻用雷神影打一场
[25-30min] 章末钩子：Eylíf 的第一条线索出现 → "她来过这里" → 驱动进入第 2 章
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

> **伏笔设计**：第 1 章中域打的是"从雷域飘来的雷霆残响"（教学 Boss），第 2 章雷域才是索尔主域——到那时玩家发现索尔已战死、群龙无首。**教学赠予与雷域正式叙事不冲突，反而埋了钩子。**

**Onboarding Checklist（对标玩家引导标准）：**
- ✅ 核心动作 30 秒内引入
- ✅ 首战必胜（教学 beat 1 无失败可能）
- ✅ 每个新机制在低风险环境引入（侵蚀演出先看、后承压）
- ✅ 至少一个机制靠探索发现（解谜 / 回溯段）
- ✅ 首章结尾留钩子（Eylíf 线索 → "再玩一章"）

### 10.3 Eylíf 定位（核心 · 已确定）

**最终定位：爱人 + 方案 A 叠加**——Eylíf 是主角生前的爱人，也战死了，比主角先一步坠入世界树根系觉醒，是她引主角进 Hoddmímir 才让他活下来，随后她自己更深入树心，压制世界树的免疫过载。

```
生前：Eylíf 是主角的爱人。两人在同一场战役中战死。
        │
诸神黄昏时，两人的灵魂一起坠入世界树根系。
        │
Eylíf 比主角先一步坠入、先一步觉醒。
她把主角的灵魂引进了 Hoddmímir —— 才让他没有消散。
（这解释了第二层契约里"主角恰好躲进 Hoddmímir"的运气）
        │
然后 Eylíf 自己走得更深，进入世界树内部，
去做一件事：压制世界树的免疫过载（按住伤口）。
        │
主角在 Hub（Hoddmímir）醒来时，Eylíf 已经不在了。
主角全程追随她留下的足迹前进——每个域都能找到她的线索。
        │
终章在建木心，主角终于追上她，发现她一直在独自硬撑。
「你守够了。换我。」—— 主角注入全部圣物之力，替她按住伤口。
```

**为什么这是最优解：**
- **私情驱动 + 因果自洽，两全**：爱人关系提供最强情感驱动；方案 A 的"她引你进 Hoddmímir"解释了主角的幸存逻辑。两者不冲突，反而互相加强。
- **情感线贯穿全程**：每个域的"Eylíf 线索"全部可用，且因为是"找回爱人"，每条线索都更揪心。
- **真相揭晓有重量**：中期揭晓"是她用自己换了你的命"，私情让这个反转更痛。
- **称呼锚点**：Eylíf 是全游戏唯一叫主角**本名**的人——她是唯一一个，在你变成"远行者 / 掠神者 / 同行者"之前，就认识"你"的人。
- **终章闭环**："你守够了。换我。"——爱人之间的接力，是整作情感顶点（见 Part 11 真结局）。

### 10.4 主要角色动机弧线（Part 5 指针的落地）

叙事的情感重量不只靠主角，每个关键角色都有自己的**动机起点 → 转折 → 落点**。这是让 NPC 像"人"而非"任务牌"的关键：

| 角色 | 动机起点 | 转折 | 落点 |
|------|---------|------|------|
| **主角** | 不想被侵蚀抹除（为自己续命） | 中点发现神也是受害者（为世界救树） | 终章用九神之力做一个选择（为一个了断） |
| **Eylíf** | 战死时想护住爱人（把主角引进 Hoddmímir） | 独自深入树心按住伤口，越撑越弱 | 主角追上她，接过她的位置——她终于能放手 |
| **引路人（无面女武神）** | 完成最后一次未竟的接引（职责） | 发现自己也在被侵蚀，接引已无意义 | 把"接引"的意义交给主角去重新定义 |
| **同命英灵 · 先行者** | 想靠自己解约（求生） | 为解约不择手段，逐渐堕落 | 成为主角的镜子——"我可能变成的样子" |
| **同命英灵 · 堕落者** | 已放弃解约、拥抱侵蚀 | 遇到还在坚持的主角，动摇 | 求一个痛快，其实还想活——主角的抉择预演 |
| **失神力旧神（弗蕾雅等）** | 困守自己的域、护住子民 | 败于主角、失去神力 | 在 Hub 旁观主角带着他们的力量前行，复杂释然 |

> **设计意图**：主角的三阶段动机（Part 5.1）不是孤立的——两个同命英灵是主角的"两条岔路预演"（先行者=堕落的可能，堕落者=放弃的可能），Eylíf 是主角的"镜像与终点"。玩家通过这些角色，提前照见自己每个选择的代价。这是让终章三选一（Part 11）真正有分量的铺垫。

### 10.5 关键 NPC 速写（武侠群像路线）

每个 NPC 像武侠小说人物——出场不多，但有来路、有去处、有执念、有软肋。

| NPC | 一句话 | 执念 | 软肋 |
|-----|--------|------|------|
| 引路人「无面女武神」 | 选中你却没能接你进殿的女武神 | 完成最后一次接引 | 她自己也快被侵蚀了 |
| 同命英灵 · 先行者 | 比你早醒、已收集数个圣物的前辈 | 想自己解约 | 不惜代价，逐渐堕落 |
| 同命英灵 · 堕落者 | 放弃解约、拥抱侵蚀的英灵 | 求一个痛快 | 其实还想活 |
| 流浪者 · 老者 | 诸神黄昏前就活着的人 | "我见过最好的世界" | 不愿承认旧世界回不来 |
| 隐藏 NPC · 维达尔 | 选择不参战、只为父复仇的沉默之神 | 想记起为何而战 | 一记起就要承担 |

## Part 11 · 终章与结局

> 本节是全部叙事线的收束——奥丁终局、建木心三真相、三条路 + 真结局。**整合自 v4.1 Part 9，并落地 Part 3 留的奥丁终局形态。所有铺垫（动机三阶段 / 同命英灵岔路 / 称呼演变 / 契约闭环）在此收口。**

### 11.1 终极战：奥丁的终局形态

> Part 3 把奥丁标为"A · 战死 / 特殊"，留了终局形态的口子——这里落地。

奥丁原典被芬里尔吞噬（战死），但作为终极 Boss，他不是以"残响"出场，而是**以"不肯赴死的意志"出场**：

- **形态**：奥丁没有完成他自己的死亡。诸神黄昏本该让他被芬里尔吞噬，但他**拒绝走完这一步**——用汲取世界树的力量，把自己吊在"将死未死"的状态里，正如他把整个世界吊在毁灭半途。**他的形态就是"黄昏未竟"这个世界状态的人格化。**
- **战斗主题**：主角打的不是一个残响、也不是一个幸存者，而是**一个执意不肯放手的老王**。他越强，世界树越虚弱——这场战斗本身就在加速世界的失血，形成"你必须快点打赢、否则世界陪葬"的压力。
- **与主角的镜像**：主角也是"该死而未死"的存在（卡在生死间的英灵）。**奥丁是主角的终极镜子**——同样卡在中间态，区别是：主角在找出路，奥丁在死守。

> **这一战的情感内核**：不是"打败大魔王"，是"两个都该死而未死的人，一个想往前走、一个死攥着过去不放"。主角赢，不是因为更强，是因为他愿意接受"该结束的要结束"。

### 11.2 建木心三真相

打穿奥丁后，主角深入建木心，三层真相依次揭开——**与动机三阶段（Part 5.1）的"困惑→抉择"完全咬合**：

| 真相 | 内容 | 呼应 |
|------|------|------|
| **真相一** | 侵蚀不是外敌，是世界树自身免疫过载（受伤太重，修复反应失控） | 印证中点反转（Part 5.3） |
| **真相二** | 神的汲取是病根（从自身免疫患者身上持续抽血，越抽越糟） | 印证"神是绝望守城者"（Part 2.4） |
| **真相三** | Eylíf 一直在世界树内压制免疫过载，快撑不住了 | 揭开 Eylíf 真相（Part 10.3） |

### 11.3 三条路 + 真结局

主角手握九件神力，在建木心面对最终抉择——**这是动机第三阶段"为一个选择"的落点**：

| 结局 | 做法 | 代价 | 情感基调 |
|------|------|------|---------|
| **补天** | 全部圣物之力注入世界树 | 所有神彻底失去神力变凡人 | 苦涩的温柔 |
| **治水** | 把侵蚀导流为受控修复力 | 世界永远带疤 | 务实的希望 |
| **蜕变** | 主角与 Eylíf 一起融入世界树 | 两人不再是"人" | 壮烈的升华 |
| **真结局** | （需完成隐藏根域）让建木新芽自己决定 | 你放弃选择权，世界自己选 | 释然的传承 |

> **三条路没有"最优解"**——每条都有得有失。这守住 P2（悲怆）：主角走到终点，握着足以改变世界的力量，却发现每个选择都要付出代价。**这正是前面两个同命英灵（Part 10.4）预演过的：先行者证明了"不惜代价"的下场，堕落者证明了"放弃"的下场，玩家在这一刻替主角做出第三种回答。**

### 11.4 真结局闭环：契约的终点

```
契约的终点：
英灵契约要求"为神效力"。
但所有神都败了 / 转化了 / 失去了神力——契约的另一方已不存在。
签约人奥丁也已被主角送去他本该去的终点。
        │
主角注入全部圣物之力，替 Eylíf 按住世界树伤口：
「你守够了。换我。」
        │
契约以"主角成为新的守护者"的形式，终极解除。
第一层契约的生存压力，在这一刻彻底闭环。
```

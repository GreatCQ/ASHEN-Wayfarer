# ASHEN 远行者 · 新方向原型对标与可行性分析

> 版本 v1.0 | 2026-06-22
> 基于"向神复仇 + 收服神 + 武器模组 + 角色连段"方向的原型游戏系统对标、可行性评估及所需改变

---

## 一、用户描述的系统核心

将用户提出的方向拆解为可对标的设计组件：

| 组件 | 描述 | 来源参考 |
|------|------|---------|
| **单自定义主角** | 类仁王2半妖，自定义形象 | Nioh 2 |
| **区域神叙事** | 每区域一个神/领主，向神复仇 | Elden Ring |
| **击败→收服** | 击败神后获得其能力，类似卡牌收集 | ENDER MAGNOLIA / Megaman |
| **角色连段** | 收服的角色提供技能，搭配组合形成连段 | 湮灭之潮骑士系统 |
| **武器模组** | 每把武器=一套完整战斗系统 | MH / Nioh 2 |
| **素材锻造** | 战斗获得的素材合成武器外观/装备 | MH |
| **养成体系** | 技能树/成长系统 | Nioh 2 / MH |

## 二、原型游戏系统对标

### 2.1 完整度排序：与用户描述系统的重合度

#### ★★★★★ 最高重合 · Nioh 2

| 用户系统 | Nioh 2对应 | 重合度 |
|---------|-----------|-------|
| 单自定义主角 | 半妖Hide，完全自定义外观 | ★★★★★ |
| 武器=战斗模组 | 11种武器×3架势=33套战斗系统 | ★★★★★ |
| 击败→获得能力 | 灵魂核心(Soul Core)：击败妖怪→获得其技能 | ★★★★★ |
| 角色搭配 | 装备2~3个灵魂核心，不同核心组合不同打法 | ★★★★ |
| 素材锻造 | 击杀掉落素材→锻造/升级装备 | ★★★★★ |
| 养成体系 | 武器技能树+忍术+阴阳术+能力值 | ★★★★★ |

**灵魂核心系统详解（最核心的对标）：**
- 击败妖怪→有概率掉落其灵魂核心
- 核心可在神社净化→装备到守护灵上
- 每个核心消耗"魂力(Anima)"施放一次妖怪技能
- 核心分主动技能+被动加成
- 可融合升级：同名核心合并→提升核心Rank（最高30）
- 核心有装备点数限制，不能全带上→**搭配取舍**

**与用户系统的差距：**
- 灵魂核心是"技能"，不是"角色"——没有叙事人格
- 没有角色间的连段combo设计（核心各自独立施放）
- 没有区域神的叙事结构

| 用户系统 | ENDER MAGNOLIA对应 | 重合度 |

**人偶系统详解：**
- 10个主要人偶，分4种攻击类型：主要(Primary)/自主(Autonomous)/冷却(Cooldown)/压迫(Pressure)

- 除战斗同伴外还有探索同伴（狼→冲刺、攀壁等）

- 没有MH式武器模组（主角本身没有独立武器系统，全靠人偶）

## 三、用户系统的独特之处

用户的系统不是简单复制某个原型，而是**四个原型的交叉组合**：

```
                    ┌── Nioh 2 ──────────────┐
                    │  武器模组(MH级深度)       │
                    │  灵魂核心(击败获能力)      │
                    │  素材锻造循环             │
                    └─────────────────────────┘
                    ┌── ENDER MAGNOLIA ───────┐
                    │  击败→收服为"角色"        │
                    │  角色有叙事人格           │
                    │  角色提供技能             │
                    └─────────────────────────┘
主角系统 =          ┌── 湮灭之潮 ─────────────┐
                    │  骑士连段combo系统        │
                    │  同步/异步搭配            │
                    │  两套编组一键切换          │
                    └─────────────────────────┘
                    ┌── 艾尔登法环 ───────────┐
                    │  区域神叙事结构           │
                    │  向神复仇/挑战            │
                    │  自由选择挑战顺序          │
                    └─────────────────────────┘
```

**这个组合在市场上没有完全相同的先例。** 最接近的是Nioh 2 + ENDER MAGNOLIA的混合体，但连段combo层面加入了湮灭之潮的骑士系统。这是创新点，也是风险点。

## 四、可行性评估

### 4.1 方向可行性：★★★★☆ 可行，但有三个关键风险

**可行性依据：**
1. 每个子系统都有成功先例 → 技术风险可控
2. "击败→收服→搭配"的循环天然支持DLC扩展 → 商业模型健康
3. MH武器模组已被验证为共斗游戏的核心引擎 → 战斗深度有保障
4. 角色连段是湮灭之潮的已验证系统 → combo可行性已证明

### 4.2 三大关键风险

#### 风险1：系统过载 — 四层系统叠加的学习曲线

| 系统层 | 学习内容 | 估计学习时长 |
|-------|---------|-----------|
| 武器模组 | 每把武器的完整招式表+派生 | 每把武器2~4h |
| 角色技能 | 14个神的技能+搭配逻辑 | 4~6h |
| 连段系统 | 角色间的combo触发条件+时机 | 3~5h |
| 锻造养成 | 素材→装备→属性→技能树 | 2~3h |

**问题：** Nioh 2本身就因系统复杂被大量玩家反映"劝退"。如果再叠加连段combo，新手学习曲线可能超过8小时才能"会玩"。

**缓解方案：** 渐进式解锁——武器模组→第一个角色→连段→锻造，每层间隔1~2小时游戏时间，不要一次性全部抛给玩家。

#### 风险2：武器 vs 角色的功能重叠

**问题：** 如果武器已经提供了完整的战斗模组（如MH那样每把武器=一套玩法），那角色技能叠加后：
- 武器负责基础输出循环，角色负责什么？
- 如果角色也是攻击技能，那和"多一把武器"有什么区别？
- 如果角色只是buff/debuff，那连段combo就做不出来

**这是设计最核心的难题。** 需要明确武器和角色的功能分工。

**推荐分工：**
- **武器 = 基础循环**（普攻/重击/闪避/架势→核心输出节奏）
- **角色 = 连段触发器**（在武器循环中插入角色技能→改变节奏/追击/控场）
- 不是"武器打+角色也打"，而是"武器搭骨架，角色填血肉"

类比音乐：武器是节拍，角色是旋律。节拍稳定循环，旋律在节拍上变化。

#### 风险3：14个"神"的叙事重量

**问题：** 每个神既是区域Boss、又是可收服角色、又有独立叙事——14个Boss × 3重身份 = 42个设计需求。ENDER MAGNOLIA只做了10个人偶就已经需要大量叙事编排，14个神的工作量更大。

**缓解方案：** 分级叙事——不是14个神都有等量叙事。核心神（4~5个）有完整故事线，边缘神（4~5个）有特色但轻叙事，隐藏神（2~3个）纯挑战向。

## 五、若延续此方向，游戏需要的改变

### 5.1 叙事结构改变

| 原方向(v2.0) | 新方向 | 变化说明 |
|-------------|-------|---------|
| 双主角(Lífþrasir×Hel) | 单自定义主角(侵蚀遗存) | 从叙事驱动转向玩法驱动 |
| 七章线性叙事 | 九域自由选择(类艾尔登法环) | 每域一个神，玩家选择挑战顺序 |
| 找恋人Eylíf | 向神复仇+寻找自我 | 主线动机从"找人"变为"找仇" |
| 劫=外部侵蚀 | 神=劫的化身 | 每个神代表劫的一种面相 |
| 三结局 | 三种对待神的态度 | 全杀/全收/选择→三种结局 |

**关键变化：Eylíf怎么办？**

Eylíf不必删除——她可以变成"最后的神"或"神的源头"：
- 所有神都在争夺Eylíf融入建木后散出的"根之声"
- 主角体内有Eylíf的回声——这是为什么神能被收服的原因
- 最终Boss不是"劫"，而是"你能不能放下Eylíf"

### 5.2 Hel的身份改变

| 原方向(v2.0) | 新方向 | 变化说明 |
|-------------|-------|---------|
| 双主角之一 | 收服的第一个"神" | 从同伴变成第一个战利品 |
| 死亡女神 | 深渊域之神 | 区域Boss身份 |
| 叙事线独立 | 教学Boss→第一个收服 | 最强教学案例 |

**这是最敏感的变化。** Hel从主角变成Boss/角色，对已有设计的冲击很大。但也有好处：
- Hel作为第一个收服的神，天然是玩家最亲近的角色
- 她的死/生双面可以做成最复杂的连段角色
- 如果Hel有完整的叙事人格，她就是连段系统最好的代言人

### 5.3 战斗系统重构

| 原方向(v2.0) | 新方向 | 变化说明 |
|-------------|-------|---------|
| 双主角切换战斗 | 武器+角色双层战斗 | 从"两个人"变成"一把武器+N个角色" |
| 侵蚀度系统 | 侵蚀度→角色共鸣度 | 侵蚀度不再只是世界状态，也是角色激活条件 |
| Boss记忆碎片 | 角色碎片+神之遗物 | 碎片升级→角色技能解锁→连段解锁 |

**新战斗架构：**

```
┌──────────────────────────────────────────────┐
│  战斗架构：武器 × 角色 × 连段 × 锻造         │
├──────────────────────────────────────────────┤
│                                              │
│  第一层：武器（基础战斗循环）                   │
│  ┌──────────┬──────────┬──────────┐          │
│  │ 大剑     │ 双刀     │ 长枪     │  ...×8~14 │
│  │ 重击循环  │ 速攻循环  │ 突刺循环  │          │
│  └──────────┴──────────┴──────────┘          │
│  武器 = 基础输出节奏，不依赖角色也能战斗        │
│                                              │
│  第二层：角色（连段触发器）                     │
│  ┌──────────┬──────────┬──────────┐          │
```

### 5.4 连段系统的设计——核心创新点

连段是本方向最核心的差异化系统。需要明确：**什么触发连段？连段怎么衔接？失败会怎样？**

#### 连段触发模型（三个方案）

**方案A：条件触发型（类湮灭之潮同步骑士）**
- 角色A技能命中→满足条件→角色B自动追击
- 玩家只需"用对角色A"，连段自动发生
- 优点：低门槛，观赏性强
- 缺点：深度有限，高玩会觉得"自动连段不够爽"

**方案B：手动衔接型（类鬼泣风格切换）**
- 角色A技能后摇中→输入角色B指令→手动衔接
- 时机正确=完美连段(加伤)，时机错误=普通释放
- 优点：深度极高，练习空间大
- 缺点：门槛极高，新手打不出来

**方案C：节奏触发型（推荐）**
- 角色A技能释放后→出现"共鸣窗口"（0.3~0.5秒）
- 在窗口内输入角色B指令=共鸣连段（加伤+特殊效果）
- 窗口外输入=普通技能释放（不强不弱）
- 优点：有操作深度但不过分，视觉反馈清晰
- 缺点：需要大量调优窗口时长

**推荐方案C**，理由：
- 湮灭之潮的同步骑士证明"条件触发"可行但深度有限
- 鬼泣的手动衔接证明"全手动"可行但门槛极高
- 节奏触发是两者的中间态，且天然适配横版游戏（横版本就强调时机）

### 5.5 武器系统设计

参考MH，推荐8~14种武器，但横版2D需要重新定义：

| 武器 | 基础循环 | 节奏关键词 | 角色连段协同 |
|------|---------|-----------|------------|
| 大剑 | 蓄力→重击→后撤 | 一击脱离 | 角色"控场"→大剑蓄力→释放 |
| 双刀 | 速攻→连击→位移 | 贴身乱舞 | 角色"上buff"→双刀全开 |
| 长枪 | 突刺→格挡→反击 | 攻防一体 | 角色"破甲"→长枪反击追打 |
| 长弓 | 蓄力→射击→翻滚 | 远程风筝 | 角色"拉近距离"→长弓蓄力射击 |
| 拳套 | 连打→蓄劲→爆发 | 近身缠斗 | 角色"浮空"→拳套空中连打 |
| 镰刀 | 大范围→吸血→位移 | 收割节奏 | 角色"群控"→镰刀范围收割 |
| 法杖 | 吟唱→释放→走位 | 预读施法 | 角色"保护"→法杖吟唱 |
| 链刃 | 中距→拉扯→切换 | 弹性距离 | 角色"固定"→链刃拉扯+打击 |

**关键设计原则：每把武器的输出循环都不依赖角色也能完成——角色是"锦上添花"不是"雪中送炭"。** 这样新手用任何武器+随便一个角色都能打，高手用特定武器+特定角色组合能打出连段。

### 5.6 角色收服设计

14个"神"的收服循环：

```
首杀（叙事Boss战）
  → 神被击败→释放其"本源碎片"
  → 本源碎片 = 角色种子（尚未可用的灰色图标）
  → 解锁：此域Boss竞技场直达

净化（叙事+玩法桥梁）
  → 在锻造台使用域素材"净化"本源碎片
  → 神以"回声"形态加入队伍
  → 获得：1个基础技能 + 角色叙事碎片(第1层)

共鸣（角色养成）
  → 使用角色技能战斗→积累"共鸣值"
  → 共鸣值满→解锁：连段协同点 + 角色叙事碎片(第2层)
  → 角色从"灰色"变为"彩色"——叙事上，神开始恢复记忆

深化（连段解锁）
  → 收集同域的精英异兽素材→强化角色
  → 解锁：第2技能 + 角色叙事碎片(第3层) + 连段衔接点
  → 角色可以与其他角色形成combo链

觉醒（终极形态）
  → 侵蚀度95%+挑战此域"侵蚀神"形态
  → 击败→获得：终极技能 + 完整叙事 + 角色专属外观
  → 角色从"回声"变为"真名"——叙事上，神完全认可你
```

## 七、结论与建议

### 7.1 方向可行，但需要明确优先级

这个方向最大的风险不是"做不出来"，而是"什么都想做导致什么都不够深"。

**建议：明确定义哪个系统是"1英尺宽1英里深"的核心引擎。**

| 系统层 | 建议深度 | 理由 |
|-------|---------|------|
| 武器模组 | **最深**（MH级） | 这是共斗游戏的命脉，浅了就不是共斗 |
| 角色收服 | **深**（ENDER MAGNOLIA级） | 这是叙事温度的来源，浅了就没有情感 |
| 连段系统 | **中深**（湮灭之潮级） | 这是差异化卖点，但不宜过度复杂 |
| 锻造养成 | **中等**（不需要MH那么深） | 太深会抢武器系统的注意力 |

### 7.2 立即需要做的三个决策

1. **Hel怎么处理？** 从主角变成第一个可收服的神，还是保留为叙事NPC？
2. **武器数量？** 8种（精简）还是14种（完整）？这直接影响开发量
3. **连段触发方式？** 方案A(自动)/B(全手动)/C(节奏窗口)？这决定战斗手感

### 7.3 如果选择这个方向，v2.0的哪些东西可以保留

| 可保留 | 需要修改 | 需要删除 |
|-------|---------|---------|
| 九域世界观 | 双主角→单主角 | 双主角切换战斗 |
| 侵蚀度系统 | 侵蚀度功能→共鸣度 | Hel作为主角的身份 |
| Boss记忆碎片 | 碎片→角色净化素材 | 七章线性叙事推进 |
| 双层地图结构 | 增加域间自由选择 | — |
| 异兽=建木免疫 | 异兽掉角色净化素材 | — |
| DLC扩展路线 | 新远行者→新神+新武器 | — |
| Eylíf情感线 | Eylíf→最后的神的真相 | 找恋人为主动机 |

---

*本文档为方向分析，所有设计细节需在确认方向后进一步细化。核心判断：此方向可行，但系统复杂度显著高于v2.0的双主角方案，需要严格管控系统叠加的学习曲线。*

## 1. 核心：双线成长体系

你提出的双线，是整个养成的骨架。**两条线物理分离，各管一摊，玩家永远清楚"要什么就去打什么"。**

```
线 A · 技能线（紧贴叙事 / 神Boss）
战胜神Boss(或剧情赠予) → 获得神器(圣物) → 幻化神影 → 复刷升级神影技能
                                                  神器 ↔ 神影 ↔ 技能 一一对应

线 B · 装备线（共斗循环 / 异兽Boss）
战胜异兽(小怪/精英/兽Boss) → 获得素材 → 合成武器+装备 → 提升基础面板
                                          独立武器系统(MH式)

两线在战斗里相乘 = 完整 Build：
武器（线B造） × 神影（线A得） = 战斗风格
```

### 1.1 两线的职责切分

| 维度 | 线A · 技能线 | 线B · 装备线 |
|------|------------|------------|
| 来源 | 神Boss（章节核心） | 异兽（小怪/精英/兽Boss） |
| 产出 | 神器 → 神影 → 主动技能 | 素材 → 武器 + 防具 |
| 决定 | "你会什么神技" | "你的武器面板多硬、节奏如何" |
| 叙事权重 | ★★★★★（贯穿主线） | ★★（玩法循环，弱叙事） |
| 获取方式 | 以战为主，剧情少量赠予 | 纯战斗掉落 |
| 对标 | 洛克人Boss武器 / ENDER MAGNOLIA同伴 | 怪物猎人素材合成 |
| 数量级 | 少而精（9个神器/神影） | 多而广（多套武器+防具） |

### 1.2 为什么这个切分好

- **认知零负担**：玩家一眼就懂——想解锁新招式去挑战神，想强化武器去刷异兽。两个目标不会互相干扰。
- **叙事与玩法各得其所**：神Boss是"叙事高潮+能力里程碑"，刻意做少做精；异兽是"日常循环+共斗内容"，可以做多做密。
- **天然支撑共斗DLC**：线B（异兽素材合成）就是怪物猎人式共斗的底座，后续多人联机狩猎异兽无缝接入。
- **神器神影一一对应**：玩家收集9个神器=集齐9个神影=9套神技，收集目标清晰可数，符合endowment effect（投入感）。

### 1.3 "剧情赠予"的设计纪律

神力以战为主，但允许剧情少量赠予——**纪律是：赠予只用在"叙事必须、战斗不合适"的节点。**

| 允许赠予的场景 | 例子 |
|--------------|------|
| 首神·教学 | 索尔之力作为教学奖励直接给（见第10节），让玩家立刻体验"神影"是什么 |
| 牺牲式交付 | 某幸存神自愿把神力传给主角（武侠传功），不打而得，强化"以战证道"反衬 |
| 终章关键 | 终章某神力作为剧情转折点赠予 |

> **赠予 ≤ 2个**（首神 + 1个叙事节点）。其余7个神器必须靠战胜获得，守住 P2「以战证道」内核。

## 2. 游戏时长与体量目标

### 2.1 竞品时长对标

| 游戏 | 主线时长 | 100%时长 | 域/区数 | 大Boss数 | 类型 |
|------|---------|---------|--------|---------|------|
| 空洞骑士 | 25-35h | 50-80h | ~14区 | 22(+8DLC) | 银河城标杆 |
| 奥日·萤火意志 | 8-12h | 15-25h | ~6区 | 7 | 平台冒险 |
| 渎神2 | 20-30h | 30-40h | ~10区 | ~15 | 硬核银河城 |
| 血污·夜之仪式 | 16-25h | 35-45h | ~10区 | 20+ | 银河城复兴 |
| 怪物猎人World | — | 200h+ | — | 37大怪 | 共斗复刷 |

### 2.2 ASHEN 的体量定位

ASHEN 是 **银河城探索 × 怪猎复刷** 的杂交品种，时长目标取两者之间偏长：

| 指标 | 目标值 | 依据 |
|------|--------|------|
| **主线时长（first ending）** | **30-40h** | 略高于空洞骑士主线，因为有双线养成 |
| **完整体验（含隐藏域+真结局）** | **60-80h** | 对标空洞骑士100% |
| **核心复刷（含神影满级+装备毕业）** | **80-120h** | 怪猎式复刷补足长尾 |
| **平均单域主线时长** | **3-4.5h** | 9主域 × 3.5h ≈ 31.5h 主线核心 |
| **单域完整探索（含支线/收集）** | **5-7h** | 含异兽刷取、隐藏路径回溯 |

> **假设：** 平均玩家单次会话 1-1.5h。每个域设计为"2-3次会话能通主线，再花1-2次会话探索/刷取"——符合横版银河城的节奏感。[PLACEHOLDER] 待纸面流程模拟与首测验证。

## 3. 九界映射：北欧为骨 · 山海经为皮

### 3.1 映射原则

- **骨（结构）= 北欧九界**：保留耳熟能详的世界框架，神是耳熟能详的北欧神，挑战才有冲击力。
- **皮（质感）= 山海经**：每个域的视觉、生态、异兽用山海经元素包裹，形成独特东方奇幻气质，避免又一个"纯北欧黑魂like"。
- **世界树 = 建木**：北欧伊格德拉希尔与山海经建木本就是同源母题（都是世界中心的通天神树），天然焊接。

### 3.2 九主域 + 二隐域 完整映射（全知名神版）

> **重要：所有章节神Boss均替换为北欧原典真实存在的知名神，原创神（炉心神/冻息/繁花母/劫·Rök）全部移除。** 每个Boss都是"玩家听过名字、且诸神黄昏里真有其人"的神，挑战才有冲击力。

|---|------------|--------------|----------------|---------|--------|--------|
| ① | **中域** | 米德加尔特 Midgard | 都广之野（建木所在·世界中心） | 黄昏余烬·人间废墟 | 教学兽（非神） | — |
| ② | **雷域** | 阿斯加德 Asgard | 雷泽（雷神居所·"雷泽有神"） | 神域天廷·雷霆 | **索尔（雷神）·首神** | ★★★ |

- **替换的4个原创神 → 知名神：** 炉心神→海姆达尔（铸域）；冻息→赫尔（寒域，冥府气质更契合）；保留弗蕾雅；劫·Rök→洛基（劫域，诸神黄昏真正的点火者）。

- **新增终极域「英灵殿废墟」：** 终极Boss奥丁不放在普通九域里，而是打完洛基后解锁的终章专属战场——瓦尔哈拉废墟，即主角当年该去却没去成的地方。**叙事闭环：你最终在"本该接纳你的英灵殿"里，向"该接纳你却没做到的奥丁"讨一个交代。**
- **隐藏神改为维达尔：** 原"遗忘王"原创神→维达尔（奥丁之子，原典中诸神黄昏后幸存、为父复仇的沉默之神）。"选择不参战"的设定与维达尔气质吻合，且是原典幸存神。

## 4. 章节叙事主轴

叙事采用 **五幕九章** 结构（幕=情绪段落，章=域）。每章一个域、一个神、一条核心主轴。

```
第一幕·苏醒（教学）
  └ 第1章 中域：我是谁，契约是什么

第二幕·追债（前期·打外围/边缘神）
  ├ 第2章 雷域：第一个神，第一份力量（索尔）
  ├ 第3章 花域：生也能吞噬人（弗蕾雅）
  └ 第4章 铸域：守桥人放弃了号角（海姆达尔）

第三幕·渡劫（中点·打华纳神族/真相反转）
  ├ 第5章 寒域：死神也在等死（赫尔）
  └ 第6章 海域：★中点真相——神不是元凶（尼约德）

第四幕·入渊（后期·打核心圈/奥丁至亲）
  ├ 第7章 巨域：旧神的尸体（提尔·战死残响）
  └ 第8章 幽域：你打了奥丁最爱的儿子（巴德尔）

第五幕·归根（终章·真凶+终极）
  ├ 第9章 劫域：放火的人早疯了（洛基·表面元凶）
  └ ★终章 英灵殿废墟：向众神之父讨一个交代（奥丁·终极Boss）

隐藏幕（探索触发）
  ├ 天域：选择不参战的复仇神（维达尔）
  └ 根域：最后的根（真结局·Eylíf）
```

### 4.1 各章核心叙事主轴

| 章 | 域 | 神 | 核心主轴（一句话） | 玩家情绪 | 动机阶段 |
|----|-----|-----|------------------|---------|---------|
| 1 | 中域 | 教学兽 | "你死了，但没死成。契约要抹掉你。" | 困惑→求生 | 觉醒 |
| 2 | 雷域 | 索尔 | "第一个神。打赢他，你才知道自己能赢神。" | 第一次强大 | 追债·起 |
| 3 | 花域 | 弗蕾雅 | "她的域很美，美得在吃人。" | 警惕·不安 | 追债·承 |
| 4 | 铸域 | 海姆达尔 | "吹响末日号角的守卫，自己先撑不住了。" | 疑惑 | 追债·转 |
| 5 | 寒域 | 赫尔 | "连掌管死亡的神，也在怕死。" | 第一次动摇 | 渡劫·疑 |
| 6 | 海域 | 尼约德 | **"神不是元凶，是溺水的人在抢木板。"** | 震动·重估 | 渡劫·悟（★中点） |
| 7 | 巨域 | 提尔 | "有的神早死了，你打的是他的怨念。" | 苍凉 | 入渊·痛 |
| 8 | 幽域 | 巴德尔 | "你打了奥丁最爱的儿子。他该出来了。" | 沉重·决意 | 入渊·宣战 |
| 9 | 劫域 | 洛基 | "放火的人早就疯了，他不是答案。" | 困惑·反高潮 | 归根·识破 |
| ★ | 英灵殿废墟 | 奥丁 | "你欠我一纸契约。你困住了整个世界。" | 决断·清算 | 归根·终结 |
| 隐1 | 天域 | 维达尔 | "一个选择不参战、只为父复仇的神。" | 释然 | 支线 |
| 隐2 | 根域 | Eylíf | "Eylíf 一直在这里。换你了。" | 升华 | 真结局 |

> **中点反转放在第6章海域**——刚好是九章的正中，符合三幕剧的中点（midpoint）规律。前五章"找神算账"的愤怒，在这里翻转为"神也是受害者"的复杂。

### 4.2 Boss 排布的三重逻辑（为什么是这个顺序）

整个 Boss 序列遵循**三条同时成立的原则**，这是"前后出现关系"的核心：

| 原则 | 含义 | 玩家体验 |
|------|------|---------|
| **① 辨识度递增** | 越往后的神越知名（索尔→…→巴德尔→洛基→奥丁） | 每次都觉得"对手更大牌了"，期待感持续走高 |
| **② 权力金字塔** | 从战场边缘神 → 华纳神族 → 奥丁至亲 → 奥丁 | 物理上越打越靠近世界权力核心 |
| **③ 原典因果链** | 每个神的位置扣住他在诸神黄昏的真实角色 | 神话粉会心一笑，普通玩家觉得顺理成章 |

**追债链条（前后关系）：** 主角是债主，要找签约人奥丁。但奥丁在最里面，必须一层层打穿他的人——

```
打他的战士（索尔/海姆达尔）
  → 打他借力的华纳盟友（赫尔/尼约德）
    → 打他的至亲（爱子巴德尔）= 向奥丁宣战
      → 打表面元凶（洛基）= 识破"放火者不是答案"
        → 够得着奥丁本人 = 终极清算
```

### 4.3 为什么终极 Boss 是奥丁

奥丁是唯一可能的终极 Boss，因为他**同时是主角所有动机的源头**：

| 身份 | 为什么指向他 |
|------|------------|
| **契约签订者** | 英灵契约是奥丁定的规矩。主角契约未履行、卡生死间被侵蚀，债主就是他 |
| **英灵殿主人** | 殿塌了没接走主角，是他失职。终章在英灵殿废墟决战，叙事闭环 |
| **不肯放手者** | 中点真相揭示神在汲取世界树续命，而奥丁是最执着"维持旧秩序、不让黄昏完成"的神。他困住了世界的重生 |
| **反高潮陷阱** | 玩家以为放火的洛基是终极Boss。打完发现：放火者早疯了，真正该负责的是那个明知世界该重生、却死攥不放的老王。比"打败大反派"更有重量 |

> **一句话：你为奥丁的战争而死，奥丁欠你一个交代，奥丁困住了世界——所有线索的终点只能是他。**

## 5. 区域 / 小区域 / 时长配置

### 5.1 三层地图结构

```
世界地图（世界树·九域挂在不同枝干）
   └ 域地图（单个域 = 银河城式互联大区）
        └ 小区域（域内的功能分区：3-5个/域）
             └ Boss竞技场（神战场 / 异兽巢穴，MH式独立场景）
```

### 5.2 单域内的"小区域"构成

每个域是一个银河城式互联地图，内部切成 **3-5 个小区域**，承担不同功能：

| 小区域类型 | 功能 | 每域数量 | 时长占比 |
|-----------|------|---------|---------|
| **入口区** | 进入域、低难度异兽、建立氛围 | 1 | ~15% |

## 10. 首神：索尔 · 新手引导落地

> 你要求："新手引导后就给玩家一个神的能力。" 这个神 = **索尔（雷神）**。下面是落地的 onboarding 设计。

### 10.1 为什么是索尔

- **全球辨识度第一**：漫威加持，雷神之锤是流行文化符号。第一个神就是索尔，冲击力拉满。
- **雷击爽快感最直接**：雷电的视觉/听觉反馈强烈，新手第一次用神影就"哇"。
- **原典战死系**：索尔在诸神黄昏与耶梦加得同归于尽——打的是"雷霆残响"，叙事上他已不在，但力量仍在锤中。符合"以战为主+教学赠予"的混合（残响给你锤，等于传功）。

### 10.2 新手引导流程（Onboarding Checklist）

```
中域·教学（第1章，1.5-2h）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[0-30s]  核心动作：移动+普攻+闪避，30秒内全introduce
[1-3min] 第一次战斗：教学异兽（必胜，无失败可能）
[5min]   首次"侵蚀"演出：让玩家看到契约侵蚀在啃噬自己 → 建立紧迫感
[10min]  解谜/平台小段：教学探索与回溯
[15min]  遇到索尔的"雷霆残响"——第一场神战（教学Boss，难度可控）
[20min]  战后：拾起 Mjölnir → 神影系统首次解锁
         → 立刻用雷神影打一场，体验"形态切换"的爽快
[25-30min] 章末钩子：Eylíf的第一条线索出现 → "她去过这里" → 驱动进入第2章
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

> **注意：** 第2章才是"雷域"（索尔的主域）。第1章中域的"索尔雷霆残响"是教学Boss——一段从雷域飘到中域的残响，作为新手的第一个神力来源。等玩家到第2章雷域，会发现索尔的完整故事（他已战死，雷域群龙无首）。这样**教学赠予**与**雷域正式叙事**不冲突，反而埋了伏笔。

### 10.3 索尔神影·教学技能规格

| 字段 | 内容 |

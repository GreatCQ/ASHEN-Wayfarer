## 一、守根人是谁？

### 一句话定义

**守根人是建木（世界树）的最后一批仆从——不是祭司，不是战士，不是英雄，是扫落叶的人。**

### 完整定义

在ASHEN的世界里，建木（世界树/尤克特拉希尔）是支撑九界的根基。建木不是一棵普通的树——它是宇宙的骨架，九界悬挂在它的枝干与根系之上。

建木会生病。

侵蚀——来自混沌之水的外部侵蚀——会像蛀虫一样啃食建木的根脉。建木的根系贯穿三口神泉，每一口泉都连接着不同的界域。侵蚀顺着泉眼侵入根系，让建木从内部开始崩解。

守根人，就是那些**替建木拔蛀虫的人**。

他们不是神选的，不是血脉传承的，不是天赋异禀的。他们是一群**自愿走到树根底下、用手去摸那道裂缝**的普通人。

他们的工作是：
- **听根**——把耳朵贴在树根上，听建木的"脉搏"，判断侵蚀从哪来、扩散到哪了
- **清创**——找到侵蚀侵入的创口，用修补之力清除坏死部分
- **缝补**——用从建木本身汲取的微量生命力，缝合裂缝
- **守望**——在裂缝修补好之前，守在那里，不让侵蚀扩散

这个工作没有荣耀。神不会来做这件事——神忙着维持自己的域。普通人不知道这件事——他们只知道世界在变差，不知道为什么。守根人在地下，在根系深处，在没人看见的地方，做着没人感谢的事。

### 守根人的座右铭

> **"根在叫。我去看看。"**

这句话来自Eylíf——第一个走进侵蚀最深处、融入建木的守根人。后来成了所有守根人的信条。

不是"我要拯救世界"，不是"这是我的使命"，而是——**"根在叫。我去看看。"**

像邻居听到隔壁有响动就去敲门一样平常。

## 二、他们为什么是守根人？

### 不是"被选中"，是"自己走过去的"

守根人最大的设定特征：**没有任何超自然的选择机制**。

- 不是血脉传承（不是父传子）
- 不是神选天命（不是神指派的）
- 不是考试选拔（没有守根人学院）
- 不是奇遇觉醒（没有被雷劈了突然开眼）

守根人的产生方式极其朴素：

**有些人天生能"听到"建木的声音。**

这种能力——"听根"——不是超能力，更接近于一种**感知敏感度**。就像有些人对气压变化敏感（关节疼），有些人对电磁场敏感（偏头痛），有些人对建木的生命脉动敏感。

这种敏感度不可遗传，不可训练，不可检测。它随机出现在人群中——可能是铁匠，可能是渔夫，可能是孩子，可能是老人。

当一个人开始"听到"建木的叫声——一种低沉的、像大地在叹息的声音——他有几个选择：

1. **忽略它**。大多数人这么做了。声音很轻，可以被当成耳鸣、风声、或者心理作用。忽略它，它会慢慢减弱，最终消失。这个人一辈子不会知道自己是守根人。

2. **被它吸引**。少数人做不到忽略。声音像一根线牵着他们，让他们不由自主地走向建木的根系——那些在地底深处、在废墟之下、在无人踏足的洞穴里的巨大根脉。他们走到根前，把手贴上去——然后他们就会**理解**。

3. **理解之后，选择留下**。建木会把它的痛苦传给他们——不是语言，是一种直接的身体感受。他们会知道：这棵树在裂，侵蚀在扩散，如果没有人在这里修补，九界会塌。

然后，**留下还是走开，是他们自己的选择。**

没有人逼他们。建木不会求他们。已经在前面的守根人不会拦他们。他们可以转身离开，回去过自己的日子。

**守根人，就是那些选择留下的人。**

### 为什么叫"守根人"？

"守根"两个字，拆开来看：

- **守**：不是"攻"，不是"护"，是"守"。守是最低姿态的——守夜人、守墓人、守山人、守灯塔的人。"守"意味着你不主动出击，你就在那里，守住一个位置，不让它失守。武侠里最强的扫地僧，他的姿态也是"守"——守住少林寺的藏经阁，守住那份清净。

- **根**：不是"树"，不是"叶"，不是"果"。根是最底层的、最看不见的、最不性感的部分。但根是一切的根基。树冠再茂盛，根烂了，树就倒了。守根人守的不是荣耀，是根基。

合在一起——**守根人**——就是一个站在最底层、守着最根本的东西的人。

这个名字本身就是一个武侠形象：一个扫地的老头，守着一座庙，庙里供着没人记得的神。但当庙要塌的时候，他是唯一抱住柱子的人。

## 三、是由谁组成的？

### 组织结构：没有组织

守根人**不是组织**。没有总部，没有层级，没有编制，没有制服，没有信物。

他们是一个**松散的、自我认定的群体**。全盛时期也不超过二十个人，散落在九界的各个角落，各自守着自己发现的那段根系。

他们之间没有联络方式——他们通过建木的根系"感知"彼此。当一个守根人在修补一段根系时，远处的守根人能感觉到根系传来的震动——"有人在干活"。仅此而已。

### 成员构成

守根人的组成完全随机，因为"听根"能力不可预测：

| 类型 | 比例 | 说明 |
|------|------|------|
| 普通人（农民/渔夫/猎手） | ~40% | 最常见的来源。生活接近土地的人，对自然变化更敏感 |
| 手艺人（铁匠/木匠/织工） | ~25% | 长期与材料打交道，对物质的结构和"伤口"有直觉理解 |
| 退伍战士/流浪者 | ~15% | 有战斗经验，在守根过程中遇到异兽时能自保 |
| 神职人员/萨满 | ~10% | 原本是神庙的侍者或部族的巫，对灵性世界有基础认知 |
| 不明身份者 | ~10% | 有些守根人来历不明，连他们自己都不记得以前是谁 |

### 历史：三个时代

**第一时代 · 拓根期（诸神黄昏之前）**

诸神黄昏之前，建木健康稳定，侵蚀几乎不存在。守根人极少——可能只有两三个人，工作很轻——偶尔清理一下根系上的寄生虫，像园丁给盆栽松土。

这个时代的守根人更像是**建木的园丁**，不是战士。

**Eylíf**就是这个时代的守根人——她是第一个"听根"能力极强的人，能直接与建木对话。她是Lífþrasir之前最伟大的守根人，也是后来融入建木、成为树之盾的那个人。

**第二时代 · 守根期（诸神黄昏爆发时）**

诸神黄昏爆发，侵蚀如海啸般涌入。建木的根系同时出现数百道裂缝。

守根人从三五人激增到近二十人——不是招募的，是侵蚀的爆发让更多人的"听根"能力被激活了。大灾变唤醒了沉睡的感知。

这是守根人最悲壮的时代：二十个人面对数百道裂缝，他们知道自己补不完。但他们一个一个地扑上去，抱住自己能抱住的那段根，用身体挡住侵蚀。

**二十个人，全部死了。**

有的死于侵蚀本身的腐蚀。有的死于被侵蚀驱狂的异兽。有的死于力竭——修补之力耗尽了他们的生命力，他们把自己"焊"在了根系上，变成了根系的一部分。

Eylíf是最后一个倒下的。她不是死去的——她是**主动融入**的。她把自己剩余的生命力完全注入建木，成为树干的一部分。她的意识散入根系，成为建木的"免疫反应"——每当侵蚀入侵，她的力量就会自动涌过去压制。

这就是为什么建木在诸神黄昏后没有完全崩塌——**因为有人用身体替它挡了一下。**

**第三时代 · 末根期（游戏开始时）**

诸神黄昏后，世界悬停在毁灭的半途。建木还在——勉强还在——因为Eylíf的融入给了它最后一口气。

但侵蚀没有停。Eylíf的力量在衰减——她不是神，她的生命力有极限。每一道她压制的裂缝，都在慢慢扩大。

新的守根人——主角——是这个时代唯一一个"听根"能力觉醒的人。

他可能是任何人。他可能以前是个铁匠，可能是个流浪者，可能是个退役士兵。某一天，他听到了那个声音——大地在叹息——然后他顺着声音走了过去。

他走到建木的根系前，把手贴上去——

他感觉到了Eylíf。

不是她的声音，是她的**疲惫**。一种撑了太久的、快要撑不住的疲惫。

然后他做出了和所有守根人一样的选择：

**他留下了。**

## 四、他们的能力有哪些？

守根人的能力分为三层：**感知层 → 修补层 → 战斗层**。从低到高，层层递进。

### 第一层：感知层（听根）

| 能力 | 说明 | 游戏内表现 |
|------|------|-----------|
| **听根** | 感知建木的生命脉动，判断侵蚀的位置、强度、扩散方向 | 小地图上侵蚀区域可见；靠近裂缝时手柄震动 |
| **辨音** | 通过根系的震动，感知远处发生的重大事件（如某个域的神在汲取力量） | 剧情触发：突然感知到某域出事了 |
| **回声感知** | 触碰建木根系时，能"回放"这段根系经历过的记忆碎片 | Boss记忆碎片系统的叙事入口——触碰Boss战场附近的根系，看到Boss的过去 |

**听根是被动能力**——守根人不需要激活它，它一直在运行。就像听力一样，你不需要"打开耳朵"，声音自然会进来。

### 第二层：修补层（清创与缝补）

| 能力 | 说明 | 游戏内表现 |
|------|------|-----------|
| **清创** | 用修补之力清除侵蚀的坏死组织，让裂缝暴露出健康的"木质" | 侵蚀区内可互动的"清创点"——清理后获得短暂安全区 |
| **缝补** | 将自身微量的生命力注入建木，缝合裂缝 | 关键剧情节点：修补域根脉的主裂缝，改变该域的侵蚀状态 |
| **引脉** | 引导建木的生命力在根系中流动，绕过被侵蚀损坏的部分——像搭桥手术 | 解锁新区域的机制：引导生命力到断裂的根系，打开通路 |
| **共鸣** | 与已收服的神的"道"共鸣，借神的力量加速修补 | 神角色技能在非战斗状态下的域效果：如繁花母的共鸣让该域植物生长，开辟新路径 |

**修补是守根人的核心能力**——不是战斗，是修补。这是他们区别于所有其他职业的根本特征。

修补之力的来源：**建木本身**。守根人不创造力量，他们是建木力量的"导管"。建木的生命力通过他们流入裂缝——他们只是那根管子。

这意味着：**修补之力对侵蚀天然有效**，因为修补之力就是建木自己的免疫反应。守根人不是在用自己的力量对抗侵蚀，是在帮建木完成它自己想完成但没力气完成的免疫反应。

### 第三层：战斗层（为什么能打）

| 能力 | 说明 | 游戏内表现 |
|------|------|-----------|
| **侵蚀感知** | 在战斗中能"看到"侵蚀在敌人身上的流动——弱点可视化 | 类似仁王2的"常暗"感知，敌人身上的侵蚀节点就是弱点 |
| **修补之刃** | 武器附带修补之力的残余——对侵蚀系敌人有额外伤害，对建木系敌人（异兽）有安抚效果 | 武器基础属性：对侵蚀系+30%伤害 [PLACEHOLDER] |
| **根脉共鸣** | 站在建木根系上方时，获得微量恢复和增益效果 | 站在根系附近时缓慢回血，鼓励玩家利用地形 |
| **侵蚀忍耐** | 守根人对侵蚀有天然抗性——不是免疫，但比普通人承受得更久 | 侵蚀度上升速度比NPC慢，但不是不上升 |
| **神的共鸣（收服后）** | 击败并收服域神后，神以"回声"形态留在守根人身边，提供战斗技能和连段触发 | 核心战斗系统：武器×角色连段 |

## 五、他们为什么有战斗力？

这是整个设定最需要解释的部分——一个"扫落叶的"为什么能打赢神？

### 核心逻辑：不是"更强"，是"更懂"

守根人的战斗力不来自力量，来自**理解**。

```
神的力量 ← 来自建木的秩序（建木越稳定，神越强）
侵蚀的力量 ← 来自混沌之水（建木越崩，侵蚀越强）
守根人的力量 ← 来自对建木的理解（每天都在触碰建木的伤口）
```

关键：**神的力量的来源（建木）正在崩塌，但守根人的力量来源（对建木的理解）反而在增强。**

因为建木越崩，裂缝越多，守根人能感知到的建木内部结构就越深。这就像一个医生——病人越病，他对这个病人的身体就越了解。当病人健康时，他只能看到皮肤表面；当病人全身是伤时，他能看到每一根血管和神经。

**所以守根人在诸神黄昏后反而比之前更强了——因为建木从未像现在这样伤痕累累，也从未像现在这样"敞开"给守根人看。**

### 具体的战斗优势

**1. 弱点感知：守根人看得到侵蚀的"纹路"**

侵蚀不是一团混沌——它有结构，有流向，有节点。普通人看侵蚀是一片黑雾，守根人看侵蚀能看到里面的"血管"。

这意味着在战斗中，守根人能精确地找到侵蚀系敌人的弱点——侵蚀流动的节点处。打这些节点，效果是打其他地方的数倍。

**2. 修补之刃：对侵蚀天然克制**

守根人的武器附带修补之力的残余。修补之力是建木的免疫反应——它天生就是用来清除侵蚀的。

对侵蚀系敌人（被侵蚀侵蚀的异兽、被侵蚀侵蚀的神），修补之刃是**针对性武器**。不是攻击力更高，是**属性克制**。

类比：你用火烧木头，效果一般；你用火烧纸，效果极好。不是因为火更强了，是因为纸对火没有抗性。

**3. 神在变弱：战力差在缩小**

诸神黄昏后，神的力量来源（建木的稳定秩序）在崩塌。每个神都在变弱——他们维持自己的域已经耗尽全力。

而守根人因为在侵蚀中行走、修补、战斗，反而在变强。

**不是守根人追上了神，是神掉下来了。**

当玩家面对一个神时，那个神已经不是全盛期的神——而是一个困守围城、力量衰减、内心恐惧的幸存者。

**4. 为什么神愿意交付力量？**

因为守根人在战斗中做了一件神做不到的事：**直接触碰侵蚀并用修补之力清除了它。**

神远离侵蚀。侵蚀消解神性——对神来说，侵蚀是致命的。他们不敢碰。

但守根人不怕。守根人没有神性可以失去。他走到侵蚀面前，把手放上去，修补——侵蚀退了。

神看到了这一幕。他们意识到：**这个凡人能做到他们做不到的事。**

不是因为他更强，是因为他更**适合**这个时代。诸神黄昏后的世界，需要的不是最强的战士，是最**对症**的人。

## 六、为什么成为主角玩家会有共鸣？

### 共鸣机制一："不是天选之子，是最后一个没走的人"

玩家对"天选之子"叙事已经审美疲劳了——从林克到士官长到龙裔，"你被选中了"已经说了太多遍。

守根人反转了这个套路：

- 你不是被选中的。你是**自己走过去的**。
- 你不是最强的。你是**最后一个还没死的**。
- 你不是天赋异禀。你是**唯一一个还愿意干的**。

"只剩我了。那我就来。"——这句话的力量在于，每个普通人都经历过这个时刻。不是英雄主义，是**责任感**——当没有人来做的时候，你站出来了，不是因为你勇敢，是因为**总得有人来**。

**武侠精神的核心就是这个**——不是因为强大才拔刀，是因为路见不平。强大是拔刀之后的事。

### 共鸣机制二："扫地僧"形象

中国玩家对"扫地僧"原型有天然的认同——那个看起来最不起眼的人，实际上是最深不可测的人。

守根人的外在形象就是"扫地僧"：
- 干最脏最累的活（清创、缝补）
- 没人知道你在干什么
- 没人感谢你
- 但你是最后一个倒下的

武侠文化里，最高境界不是"天下第一"，是"隐于市"——你知道自己很强，但你选择做最卑微的事。守根人就是这个形象。

### 共鸣机制三："努力提升自己"的代入感

玩家提问里明确说了："能让玩家一下子就有带入感，认同这种我努力提升自己拯救世界的设想。"

守根人的成长曲线天然支持这种代入：

- **开局弱**：守根人不是战士出身，开局战斗能力很弱。他只有听根和修补——这些在战斗中几乎没有直接用途。
- **成长合理**：他的变强不是"升级加属性点"，是**理解加深**——每修补一道裂缝，他对建木的理解就深一分，能调用的建木力量就多一点。这是"修炼"的武侠逻辑——不是系统给你加数值，是你对"道"的理解更深了。
- **收服神 = 突破瓶颈**：每次击败并收服一个神，相当于武侠里的"奇遇"——前辈传功，你的境界跃升。不是你突然变强了，是你借到了更高层次的力量。
- **最终战力来源多元**：武器（自己练的）+ 神的共鸣（前辈传的）+ 修补之力（对道的理解）= 三源合一。这符合武侠里"内功+外功+兵器"的三合一体。

### 共鸣机制四：逆天改命

玩家说的"逆天改命"，在守根人身上是这样体现的：

- **天命说**：诸神黄昏不可逆转，世界注定毁灭——这是北欧神话的"天命"
- **守根人说**：我不信。树还在，根还在，我就还有机会——这是武侠式的"不服"

但这个"不服"不是无脑热血——守根人知道自己在做一件可能徒劳的事。他可能修补不了。建木可能还是会倒。但他还是做了。

**"不是因为一定能成功才去做，是因为不做就一定不会成功。"**

这就是逆天改命的本质——不是结果论（你改了命），是过程论（你敢去改）。

## 七、他们在原神话里面是否存在？

### 直接回答：不存在。但每个零件都有神话根基。

"守根人"这个名称、这个组织、这个设定，在北欧原典和中国原典中都不存在。它是ASHEN的原创概念。

但是——它的每一个组成部分，都能在神话中找到对应的原型：

### 北欧神话根基

| 守根人要素 | 北欧神话原型 | 对应关系 |
|-----------|-----------|---------|
| 替建木浇水、修补裂缝 | **诺伦三女神（Norns）** | 诺伦女神每天用兀儿德之泉的水浇灌世界树的根，维持树木健康。守根人是"人类版的诺伦" |
| 听根——感知建木的状态 | **密米尔（Mímir）** | 密米尔守在智慧泉旁，他是"知道最多的人"。守根人的"听根"是密米尔的"智慧"的平民版 |
| 人类与建木的天然连接 | **阿斯克与恩布拉（Ask & Embla）** | 北欧神话中，第一对人类由神用白蜡木和榆木创造。人类本身就是"树木所化"——与建木有天然的血缘 |
| 每个农庄的守护树 | **Vårdträd（守护树传统）** | 维京时代每个农庄都有一棵"守护树"（vårdträd/tuntre），被视为家族的灵魂锚点。守护树是世界树的微缩版。农人会照料它、供奉它——这就是"守根"的历史实践 |
| 侵蚀啃食建木的根 | **尼德霍格（Níðhöggr）** | 毒龙尼德霍格和无数蛇类盘踞在世界树根部，不停啃食树根。守根人对抗的"侵蚀"就是尼德霍格的啃食 |
| 建木脆弱、需要持续维护 | **原典描述** | 《埃达》多次描述世界树"承受着痛苦"——被鹿啃叶、被龙啃根、被蛇缠干。世界树从来不是无敌的，它需要被照顾 |

**最关键的根基：Vårdträd（守护树传统）**

这是真实存在的历史实践——维京时代的北欧人真的会在自家农庄种一棵"守护树"，照料它、供奉它、绝不砍伐它。这棵树是世界树（Yggdrasil）的微缩版，代表家族的命运。

**守根人，就是那些把"照料自家守护树"这件事做到极致的人——他们守护的不是自家门前那棵小树，是整个宇宙的那棵大树。**

### 中国神话根基

| 守根人要素 | 中国神话原型 | 对应关系 |
|-----------|-----------|---------|
| 建木 = 世界树 | **建木（《山海经》）** | 《山海经·海内经》记载的建木，"青叶紫茎，玄华黄实，百仞无枝"，是沟通天地的天梯。ASHEN的世界树直接用了"建木"这个名字 |
| 守根人 = 沟通建木的人 | **巫（萨满）** | 古蜀文明中，巫师是人与神之间的沟通者。三星堆青铜神树就是巫师用来"上天"的工具。守根人是"建木的巫"——不是用建木上天，而是替建木修补 |
| 绝地天通 = 诸神黄昏 | **颛顼断建木** | 传说颛顼砍断建木，隔绝天地，史称"绝地天通"。这与诸神黄昏"诸神与人类的连接断裂"完美对应 |
| 修补之力 | **女娲补天** | 女娲炼五色石补天——这是中国神话中"修补世界"的原型。守根人的核心行为就是"补"——补建木的裂缝 |
| 守根人的精神 | **大禹治水** | 大禹不是神，是凡人。他治水不是靠法力，是靠"三过家门而不入"的执着。守根人"修补建木"的精神内核与大禹治水完全一致——普通人做不普通的事，靠的不是天赋，是不放弃 |

**最关键的中国根基："补"的文化基因**

中国文化里有一种独特的"修补"传统——不是推翻重建，是缝补修复：

- 女娲补天——天塌了，补
- 大禹治水——水来了，导
- 精卫填海——海太大了，填
- 愚公移山——山挡路了，挖

这些故事的共同特征：**主角不是最强的，但ta是最轴的。** 别人说不可能，ta说"我就要干"。不是因为一定能成功，是因为不干不行。

**守根人就是这个文化基因在北欧神话皮肤下的实现。**

### 总结：守根人的神话DNA

```
守根人 = 诺伦女神的职责（浇灌世界树）
       + Vårdträd传统的历史实践（农庄守护树）
       + Ask/Embla的血脉连接（人类=树木所化）
       + 巫的沟通角色（人神之间的桥梁）
       + 女娲/大禹的修补精神（凡人修补世界）
       + 扫地僧的武侠意象（最卑微的人最伟大）
```

每一个零件都有出处，但组合是全新的。这种"旧零件新组装"的方式，既保证了神话根基的厚重感，又给了玩家新鲜感——他们能隐约感觉到"这个设定有根基"，但不会觉得"这不就是XX神话的翻版吗"。

## 八、玩家是否对守根人这个设定非常清楚？

### 认知度评估：中等偏低，需要通过游戏内引导建立

**坦诚说："守根人"不是一个玩家一听就懂的词。**

"猎人"（MH）一听就懂——打猎的。
"褪色者"（艾尔登法环）一看就知道——颜色褪了的人。
"半妖"（仁王2）——一半是妖。
"不死人"（黑魂）——不会死的人。

"守根人"呢？——守着根的人？什么根？树根？

这个词需要解释。但这不是问题——**大多数好的主角身份设定都需要在游戏内建立认知**。关键是要在玩家接触的前5分钟内，让他们"get"到这个身份的核心。

### 认知建立策略：30秒法则

**目标：玩家在操作主角的前30秒内，不需要任何文字说明，就能理解"守根人是干什么的"。**

#### 开场30秒设计

```
[画面] 黑屏。低沉的声音——像大地在叹息。
[字幕] "根在叫。"
[画面] 淡入。主角站在一条巨大的树根旁。树根有一道裂缝，
       黑色的侵蚀从裂缝中渗出。
[操作提示] 按R2触碰树根。
[玩家操作] 主角把手贴上树根。
[画面] 裂缝被一层温暖的光缝合。主角微微喘息。
[字幕] "又一道裂缝。越来越多了。"
[画面] 主角站起来，拿起身边的武器，向远处走去。
```

**这30秒传达了什么：**
1. "根在叫"——建木在出问题
2. 主角能感知到——他有特殊能力
3. 触碰树根能修补——这是他的核心行为
4. 修补消耗体力——这不是无代价的
5. 裂缝越来越多——情况在恶化
6. 他拿起武器——他不只会修补，他还要战斗

**不需要任何旁白解释"守根人是什么"。玩家通过操作，自己就理解了。**

#### 后续5分钟：通过NPC补全认知

第一个NPC（可能是某个域的幸存者）看到主角修补树根后：

> "你是……守根人？我以为你们都死了。"

这句台词同时完成了三件事：
1. 告诉玩家这个身份叫"守根人"
2. 暗示守根人曾经存在，但已经几乎灭绝
3. 暗示主角是"最后的守根人"——稀缺感和使命感

#### 后续30分钟：通过世界叙事深化

通过环境叙事——废弃的守根人营地、刻在树根上的修补痕迹、死去的守根人的遗物——让玩家拼凑出守根人的历史。

**不要用长篇旁白讲设定。让玩家自己发现。**

### 认知建立的三个层次

| 层次 | 时机 | 方式 | 目标 |
|------|------|------|------|
| **本能认知** | 前30秒 | 操作体验 | "这人能修树，树在坏，他在帮忙" |
| **身份认知** | 前5分钟 | NPC台词 | "他叫守根人，守根人快死光了" |
| **精神认知** | 前30分钟 | 世界叙事 | "守根人不是被选的，是自己留下的。都死了。只剩他了。" |

当玩家在30分钟内走完这三个层次，"守根人"这个词就不再是一个需要解释的设定——它变成了**玩家的自我认同**。

### 文化认知差异

| 市场 | 对"守根人"的直觉理解 | 需要补强的部分 |
|------|-------------------|-------------|
| **中国** | "守X人"是熟悉的职业格式（守夜人/守山入/守墓人）。"根"对应建木/根脉/祖根。武侠"扫地僧"意象天然共鸣 | 需要补强北欧神话根基 |
| **日本** | "守"文化（守護者）+"根"（霊根/木の根）。"最後の一人"原型高度成熟（MH/魂系列/尼爾） | 需要补强建木的世界观意义 |
| **欧美** | "Keeper"是熟悉的奇幻职业格式（Root Keeper/Warden）。"最后的守卫"原型成熟（指环王/巫师/魂系列） | 需要补强"修补"概念——西方更习惯"毁灭/重建"而非"缝补" |

**欧美市场的特殊处理**：西方奇幻传统中，英雄通常是"推翻旧秩序建立新秩序"（亚瑟王/指环王/魂系列），而守根人是"缝补旧秩序让它别倒"——这个概念对欧美玩家需要额外引导。建议在英文版中使用 **"The Mender"** 或 **"Rootwarden"** 而非直译，并在前期通过视觉叙事强调"修补"行为。

## 九、守根人详细设定

### 9.1 守根人能力总表

```
┌─────────────────────────────────────────────┐
│              守 根 人 能 力 体 系              │
├─────────────────────────────────────────────┤
│                                             │

│  ├─ 听根：感知建木脉动/侵蚀位置                │

│                                             │
│  修补层（主动·修补）                          │
│  ├─ 清创：清除侵蚀坏死组织                    │

│  ├─ 引脉：引导建木生命力搭桥开路               │

│                                             │

│  ├─ 侵蚀释放：主动释放体内侵蚀，大幅增强战力    │

│  └─ 融根：将自身完全融入建木，成为树的一部分    │
│           意识散入根系，肉体消失               │
│           Eylíf的结局。真结局路线的最终选择     │
└─────────────────────────────────────────────┘
```

### 9.2 守根人的"禁术"：侵蚀释放与融根

**侵蚀释放**

守根人体内天然有微量的侵蚀残留——因为长期接触侵蚀环境。正常情况下，修补之力会压制这些残留。

但在极端战斗中，守根人可以**主动释放**这些侵蚀——让侵蚀的力量暂时为自己所用。

效果：大幅增强战斗能力（侵蚀形态），但代价是**侵蚀度永久上升一小段**，不可逆转。

这是"用你要对抗的东西来战斗"的经典设计——你每次使用禁术，都在变成你想要消灭的东西。

**融根**

守根人的最终禁术——也是Eylíf的选择。

将自己的全部生命力注入建木，肉体融入树干，意识散入根系。守根人不再是"人"，成为建木的一部分。

效果：极大增强建木的稳定性，相当于给建木注入一剂强心针。但守根人作为"人"的存在消失了。

这是真结局路线的核心选择——玩家可以选择融根，成为新的Eylíf，替世界树再撑一个纪元。

### 9.3 守根人的武器与战斗风格

守根人的武器不是传统兵器——它们是**修补工具的武器化**。

| 武器类型 | 原始用途 | 战斗风格 | 类比 |
|---------|---------|---------|------|
| 根刀 | 切除侵蚀坏死组织的手术刀 | 快速、精准、连击 | 太刀（MH） |
| 缝针 | 缝合裂缝用的巨型针 | 中速、穿刺、抓取 | 枪（MH） |
| 锤凿 | 凿开侵蚀硬壳的工程锤 | 慢速、重击、破防 | 大锤（MH） |
| 藤鞭 | 引导生命力流动的藤蔓 | 远程、控制、范围 | 长枪/操虫棍 |
| 根盾 | 临时搭建的根系屏障 | 防御、反击、阵地 | 长枪+盾（MH） |
| 双锉 | 磨平侵蚀表面的手持锉 | 双持、快速、弱单发 | 双刀（MH） |

**设计原则：每把武器的"战斗动作"和"修补动作"是同一套动作模组。**

这意味着守根人战斗时用的不是"武术"，是"修补术"——他的斩击是清创动作的加速版，他的穿刺是缝补动作的暴力版，他的重击是凿击动作的战场版。

这让守根人的战斗有一种独特的美学：**不是武术家在打拳，是工匠在工作——只是工作的对象从树根变成了敌人的身体。**

### 9.4 守根人的成长体系

```
成长来源（三源合一）
├─ 修补深度（内功）
│   每修补一道裂缝 → 对建木的理解+1 → 修补之力效率+X%
│   累积到一定深度 → 解锁新的引脉路径（探索能力）
│
├─ 武器熟练度（外功）
│   每使用某武器战斗 → 该武器熟练度+1 → 解锁新动作/连段
│   类似MH的武器精通，每种武器有独立成长树
│
└─ 神的共鸣（奇遇/传功）
    每收服一个神 → 获得该神的"道" → 解锁新技能/连段触发器
    神的等级影响共鸣强度 → 可通过重复挑战Boss（侵蚀度复刷）提升
```

### 9.5 守根人与其他角色的关系矩阵

| 角色 | 与守根人的关系 | 互动方式 |
|------|-------------|---------|
| **Eylíf** | 前辈·恋人·建木之盾 | 通过根系回声沟通；她的意识碎片散在建木中，是记忆碎片系统的核心 |
| **域神（7位）** | 对手→传功者 | 首战是Boss战；击败后通过"证明→交付"获得神的回声 |
| **隐藏神（2位）** | 最深层的试炼 | 需要特殊条件触发；交付后提供最强共鸣 |
| **枢纽NPC（4位）** | 情报与补给 | 不参与战斗，提供域信息、侵蚀预警、武器维护 |
| **域内NPC（9位）** | 任务与世界观 | 每域1位核心NPC，提供该域的历史和神的背景 |
| **流浪NPC（3-4位）** | 意外与变数 | 在域之间游走，可能提供跨域线索或隐藏任务 |
| **环境NPC（5-8位）** | 氛围与 immersion | 不直接互动，通过环境叙事传达信息 |

### 9.6 守根人的视觉设计方向

**核心矛盾：卑微感 vs 力量感**

守根人的视觉设计要在"看起来像个普通劳动者"和"实际上是个能跟神打的狠角色"之间找平衡。

| 元素 | 设计方向 | 理由 |
|------|---------|------|
| 体型 | 中等偏瘦，不魁梧 | 不是力量型角色，是技巧型 |
| 服装 | 带兜帽的工作服，有修补痕迹 | 劳动者的外观，兜帽增加神秘感 |
| 手部 | **手掌有树皮纹路/木质化痕迹** | 长期接触建木的身体变化——核心视觉标识 |
| 眼睛 | 瞳孔中有微弱的绿色光芒 | "听根"能力的视觉表现——不张扬但能注意到 |
| 武器 | 工具改造的武器，不是正统兵器 | 强化"工匠"身份，区别于"战士" |
| 装饰 | 极简，可能有建木碎片挂件 | 不追求华丽，实用主义 |
| 侵蚀痕迹 | 身体某些部位有黑色侵蚀纹路 | 长期接触侵蚀的代价——随着游戏推进会扩散 |

**手部的木质化是最重要的视觉标识。** 当玩家看到这个角色的手——不像人手，像树根——他们立刻就知道：这个人和树有某种深层联系。这比任何文字说明都有效。

### 9.7 守根人的叙事弧线

```
第一幕：萌芽（中域）
  "根在叫。"
  主角觉醒听根能力 → 发现第一道裂缝 → 修补 → 遇到第一个域神
  玩家认知建立：我是谁，我能做什么，我要面对什么

第二幕：行远（南/东/北域·并行）
  "原来不只是我的根在叫。"
  主角发现九界的根系都在一起颤动 → 前往三个域 → 每个域一个神
  玩家成长：每收服一个神，对建木的理解深一层

第三幕：渡劫（海内域·中点反转）
  "侵蚀不是敌人。"
  主角发现侵蚀的本质是建木的免疫过载 → 认知反转
  神不是在"对抗"侵蚀，是在"加剧"它（通过汲取建木力量）
  玩家选择：继续打神？还是改变策略？

第四幕：入渊（西/幽域·并行）
  "神不是敌人。方法才是。"
  主角深入两个隐藏域 → 发现诸神黄昏的真相 → 遇到隐藏神
  玩家理解：守根人不是在"打败"神，是在"说服"神松手

第五幕：归根（大荒域→建木心）
  "只剩我了。那我就来。"
  最终决战：不是打Boss，是做一个选择
  三结局：补天/治水/蜕变 + 真结局（融根/让新芽自己决定）
```
